# Volunteer Connect (site023)

## 개요
- **사이트 이름:** Volunteer Connect
- **포트 번호:** 9352
- **주제:** 봉사활동 모집 서비스

## 주요 기능
- 지역별 봉사활동 모집 리스트 탐색 및 검색
- 봉사활동 상세 정보 확인 및 참여 신청
- 참여 파트너 기관 목록 및 정보 조회
- 개인별 봉사 신청 내역 실시간 확인 및 관리
- 신청 단계별 상태 알림 기능

## 의도된 오류 (3개)
1. **site023-bug01 (DB 신청 중복 검증 누락):** 사용자가 이미 신청한 동일한 봉사활동에 대해 서버 측 중복 체크 로직이 없어, 여러 번 신청을 눌러 중복 데이터가 계속 생성되는 논리적 오류가 있습니다.
2. **site023-bug02 (잘못된 API 상태 코드):** 봉사활동 모집이 이미 마감(`Closed`)된 상태에서도 신청을 보내면, 실제로는 저장이 되지 않음에도 서버가 성공 코드(`201 Created`)를 반환하여 사용자를 오인하게 만드는 결함이 있습니다.
3. **site023-bug03 (관리자 데이터 노출):** 기관 목록 조회 API에서 일반 사용자에게 공개되어서는 안 되는 관리자 이메일(`adminEmail`)과 내부 관리 메모(`adminMemo`) 필드가 응답 JSON에 그대로 포함되어 노출되는 보안 취약점이 있습니다.
