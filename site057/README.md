# ClassNote Share (site057)

## 개요
- **사이트 이름:** ClassNote Share
- **포트 번호:** 9386
- **주제:** 강의 노트 공유 서비스

## 주요 기능
- 경제학, 컴퓨터공학 등 다양한 과목의 강의 노트 탐색 및 다운로드
- 인기 노트를 기반으로 한 학업 우수 자료 큐레이션
- 나의 학습 노트를 PDF, DOCX 형식으로 공유 및 업로드
- 자료별 댓글 소통을 통한 질의응답 및 학습 커뮤니티 기능
- 아카데믹 인디고와 라이트 페이퍼 컬러를 활용한 지적이고 정돈된 UI

## 의도된 오류 (3개)
1. **site057-bug01 (DB 댓글 연결 오류):** 댓글 저장 시 `noteId` 필드에 실제 노트 ID가 아닌 작성자의 `userId`가 잘못 저장됩니다. 이로 인해 내가 작성한 댓글이 엉뚱한 곳에 표시되거나 목록에서 누락되는 관계 결함이 있습니다.
2. **site057-bug02 (네트워크 파일 타입 검증 누락):** 업로드 API에서 허용되지 않은 파일 확장자(예: .exe)에 대한 검증이 누락되어, 유효하지 않은 파일도 업로드 성공 메시지를 반환하는 검증 결함이 있습니다.
3. **site057-bug03 (비공개 노트 접근 실패):** 작성자가 비공개(`isPrivate: true`)로 설정한 노트임에도 불구하고, 해당 노트의 고유 ID를 알고 직접 접근할 경우 소유권 확인 없이 내용이 노출되는 보안 취약점이 존재합니다.
