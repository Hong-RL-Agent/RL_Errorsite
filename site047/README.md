# BoardGame Cafe (site047)

## 개요
- **사이트 이름:** BoardGame Cafe
- **포트 번호:** 9376
- **주제:** 보드게임 카페 예약 서비스

## 주요 기능
- 스플렌더, 테라포밍 마스 등 다양한 보드게임 목록 조회 및 난이도/카테고리별 탐색
- 실시간 테이블 현황 확인 및 원하는 시간대별 예약 시스템
- 개인별 예약 이력 관리 및 방문 일정 확인
- 나무 질감의 아늑하고 따뜻한 우드 톤의 카페 전용 UI
- 최근 시스템 로그를 통한 실시간 예약 모니터링 기능 (Bug 포함)

## 의도된 오류 (3개)
1. **site047-bug01 (DB 테이블 중복 예약):** 서버 측에서 동일 시간대의 테이블 점유 여부를 검증하지 않아, 같은 테이블에 서로 다른 여러 명의 예약이 동시에 성공할 수 있는 동시성 결함이 있습니다.
2. **site047-bug02 (네트워크 오래된 데이터):** 테이블 현황 API가 예약 성공 후에도 최신 상태를 즉시 반영하지 않고 이전의 '예약 가능' 상태를 계속 반환하여 사용자에게 혼란을 주는 결함이 있습니다.
3. **site047-bug03 (관리자 예약 목록 노출):** 최근 예약 현황을 보여주는 API 응답에 일반 고객들의 전화번호와 상세 예약 정보 등 민감한 관리자용 데이터가 여과 없이 포함되어 전송되는 보안 취약점이 있습니다.
