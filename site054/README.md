# GiftBox Picker (site054)

## 개요
- **사이트 이름:** GiftBox Picker
- **포트 번호:** 9383
- **주제:** 선물 추천 및 위시리스트 서비스

## 주요 기능
- 예산별, 대상별(친구, 연인, 부모님 등) 맞춤 선물 탐색 및 추천
- 마음에 드는 선물을 위시리스트에 담고 관리하는 기능
- 브랜드 및 상품명 검색을 통한 실시간 선물 필터링
- 캔디 핑크와 소프트 라벤더 컬러를 활용한 화사하고 매력적인 라이프스타일 UI
- 상세 모달을 통한 상품 정보 확인 및 소셜 공유 시뮬레이션

## 의도된 오류 (3개)
1. **site054-bug01 (DB 예산 필터 오류):** 최대 예산 필터링 시 `이하(<=)`가 아닌 `이상(>=)` 조건이 적용되어, 사용자가 설정한 예산보다 비싼 고가의 상품들만 목록에 노출되는 비논리적 결함이 있습니다.
2. **site054-bug02 (API 응답 포맷 불일치):** 위시리스트 API가 JSON 배열 객체가 아닌, 배열 전체를 한 번 더 문자열화(`JSON.stringify`)하여 반환합니다. 이로 인해 클라이언트에서 정상적인 배열 처리가 불가능해지는 데이터 형식 결함이 있습니다.
3. **site054-bug03 (위시리스트 소유권 검증 누락):** 특정 사용자의 위시리스트를 조회하는 API에서 요청자의 소유권 확인이 누락되어, `wishlistId` 조작을 통해 타인의 비공개 위시리스트 정보를 무단으로 열람할 수 있는 보안 취약점이 존재합니다.
