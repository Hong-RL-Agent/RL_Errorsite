# Laundry Now (site024)

## 개요
- **사이트 이름:** Laundry Now
- **포트 번호:** 9353
- **주제:** 무인 세탁소 기기 사용 현황 서비스

## 주요 기능
- 세탁기 및 건조기 실시간 사용 현황(사용 가능, 사용 중) 확인
- 기기별 이용 요금 정보 제공
- 원하는 기기 사전 예약(15분 대기) 기능
- 과거 세탁기/건조기 이용 내역 조회
- 지점별 기기 요약 정보 대시보드

## 의도된 오류 (3개)
1. **site024-bug01 (DB 상태 동기화 오류):** 사용자가 기기를 예약했음에도 불구하고, 서버 측에서 해당 기기의 상태를 'reserved'로 업데이트하지 않아 현황판에 여전히 '사용 가능'으로 표시되는 논리적 결함이 있습니다.
2. **site024-bug02 (네트워크 응답 누락):** 기기 목록 API 응답 시 특정 기기(주로 짝수 ID)의 `status` 필드가 누락되어 전송됩니다. 이로 인해 화면에서 해당 기기의 상태가 'UNKNOWN'으로 표시되는 현상이 발생합니다.
3. **site024-bug03 (사용자 기록 접근 제어 실패):** 이용 기록 조회 API에서 사용자의 권한을 검증하지 않아, `userId` 쿼리 파라미터만 변경하면 타인의 민감한 이용 내역(날짜, 금액, 특이사항 등)을 무단으로 조회할 수 있는 IDOR 취약점이 있습니다.
