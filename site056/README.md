# DeskPlant Office (site056)

## 개요
- **사이트 이름:** DeskPlant Office
- **포트 번호:** 9385
- **주제:** 사무실 식물 구독 관리 서비스

## 주요 기능
- 몬스테라, 고무나무 등 오피스 공기 정화용 식물 구독 플랜 탐색
- 난이도별(Easy, Medium, Hard) 맞춤형 식물 필터링 및 상세 정보 제공
- 정기 배송 일정 확인 및 배송일 변경 신청 기능
- 나의 구독 현황 관리 및 월 구독료 요약 확인
- 오피스 그린과 소프트 그레이 컬러를 활용한 깔끔하고 현대적인 B2B 구독 서비스 UI

## 의도된 오류 (3개)
1. **site056-bug01 (DB 구독 중복 생성):** 동일한 사용자가 이미 이용 중인 구독 플랜을 중복으로 신청할 때 이를 검증하지 않아, 서버에 같은 구독 데이터가 여러 개 생성되는 결함이 있습니다.
2. **site056-bug02 (네트워크 실패 메시지 누락):** 배송일 변경 요청이 과거 날짜 설정 등의 이유로 실패할 때, 서버가 구체적인 에러 메시지 없이 빈 응답을 반환하여 클라이언트에서 사용자에게 실패 원인을 안내하지 못하는 결함이 있습니다.
3. **site056-bug03 (결제 금액 검증 누락):** 구독 신청 시 클라이언트에서 보낸 구독료(`monthlyPrice`)를 서버 측 원본 가격과 대조하지 않고 그대로 저장하여, 가격 변조 공격에 취약한 보안 결함이 존재합니다.
