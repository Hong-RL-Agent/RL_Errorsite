# Campus Meal Vote (site048)

## 개요
- **사이트 이름:** Campus Meal Vote
- **포트 번호:** 9377
- **주제:** 학식 메뉴 투표 서비스

## 주요 기능
- 학생회관, 중앙식당 등 교내 식당별 오늘의 메뉴 조회 및 검색
- 선호하는 메뉴에 대한 실시간 투표 및 인기 메뉴 순위 확인
- 메뉴별 칼로리 및 상세 영양 정보 확인 기능
- 투표 결과의 시각화 그래프 제공 (실시간 통계)
- 캠퍼스 네이비와 오렌지 컬러를 활용한 활기차고 모던한 아카데믹 UI

## 의도된 오류 (3개)
1. **site048-bug01 (DB 투표 중복 저장):** 동일 학생이 동일한 날짜의 메뉴에 대해 중복 투표를 하는지 검사하는 로직이 없어, 한 명이 무제한으로 투표를 반복하여 결과를 왜곡할 수 있는 결함이 있습니다.
2. **site048-bug02 (네트워크 응답 스키마 오류):** 투표 결과를 반환하는 API에서 약속된 숫자형 `percentage` 필드 대신 문자열형 `percentText` 필드를 반환하여, 클라이언트의 그래프 렌더링 로직이 정상 작동하지 않는 결함이 있습니다.
3. **site048-bug03 (사용자 인증 우회):** 정식 로그인 절차 없이 단순히 요청 헤더에 임의의 `x-student-id` 값을 포함하기만 하면 인증된 사용자로 간주하여 투표를 허용하는 보안 취약점이 존재합니다.
