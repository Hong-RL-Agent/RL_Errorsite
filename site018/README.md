# JobStarter (site018)

## 개요
- **사이트 이름:** JobStarter
- **포트 번호:** 9347
- **주제:** 신입 개발자 채용 공고 탐색 서비스

## 주요 기능
- IT 기업 채용 공고 목록 조회 및 페이지네이션
- 직무(백엔드/프론트엔드) 필터링 기능
- 관심 있는 채용 공고 북마크 저장
- 본인의 지원 현황 상세 조회
- 이력서 준비 체크리스트 관리
- 회사 정보 상세 모달 뷰

## 의도된 오류 (3개)
1. **site018-bug01 (DB 페이징 오류):** 페이지네이션 시 `offset` 계산 로직의 결함으로 인해, 다음 페이지로 넘어갈 때 이전 페이지의 데이터가 중복되어 나타나거나 특정 데이터가 누락되는 현상이 발생합니다.
2. **site018-bug02 (네트워크 요청 파라미터 손실):** 직무 필터 선택 시 클라이언트가 보낸 필터 조건(`position`) 파라미터를 서버 측에서 무시하여, 필터가 적용되지 않은 전체 목록이 반환됩니다.
3. **site018-bug03 (비인가 지원 현황 조회):** 지원 현황 상세 조회 API에서 권한 검증이 누락되어, 타인의 `applicationId`를 알면 개인정보와 합격 여부 등 민감한 정보를 무단으로 열람할 수 있는 IDOR 취약점이 있습니다.
