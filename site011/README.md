# Garden Planner (site011)

## 개요
- **사이트 이름:** Garden Planner
- **포트 번호:** 9340
- **주제:** 홈가드닝 식물 관리 서비스

## 주요 기능
- 내 식물 목록 조회 및 검색/필터링
- 식물 상세 정보 확인 및 물주기 체크
- 식물 성장 기록 관리 (키, 메모)
- 가드닝 팁 제공

## 의도된 오류 (3개)
1. **site011-bug01 (DB 상태 저장 누락):** 물주기 버튼 클릭 시 응답은 성공으로 오지만 서버 상의 상태값이 변하지 않아 새로고침하면 다시 물을 주지 않은 상태로 돌아옵니다.
2. **site011-bug02 (API 응답 스키마 불일치):** 성장 기록 조회 시 서버가 배열이 아닌 `{data: [...]}` 객체로 응답하여 프론트엔드에서 렌더링 에러가 발생합니다.
3. **site011-bug03 (사용자 식별 검증 누락):** 성장 기록 추가 시 서버가 요청 본문의 `ownerId`를 그대로 신뢰하여 보안에 취약합니다.
