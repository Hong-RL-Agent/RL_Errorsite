# PetCare Hub (site004)

## 개요
- **사이트 이름:** PetCare Hub
- **사이트 ID:** site004
- **포트 번호:** 9333
- **기술 스택:** Node.js, Express, Vanilla HTML/CSS/JS

## 실행 방법
```bash
cd site004
npm install
npm start
```
서버가 실행되면 브라우저에서 `http://localhost:9333` 으로 접속할 수 있습니다.

## API 엔드포인트 목록
- `GET /api/health` - 서버 상태 확인
- `GET /api/pets` - 반려동물 프로필 목록 조회
- `GET /api/vaccines` - 예방접종 일정 조회 (필터 옵션 지원)
- `GET /api/appointments` - 병원 예약 내역 조회
- `POST /api/appointments` - 병원 예약 생성
- `GET /api/records` - 건강 기록 조회
- `POST /api/records` - 새 건강 기록 추가

## 정상 작동 기능 목록 (인터랙션)
1. **반려동물 프로필 데이터 조회 및 표시**
2. **반려동물 선택 (UI 업데이트 및 연동 데이터 갱신)**
3. **다가오는 예방접종 일정 필터 적용** (버그 연동)
4. **건강 기록 조회 및 새 기록 추가 폼 제출**
5. **사료 추천 새로고침** (클라이언트 로직)
6. **병원 검색 및 예약 폼 제출** (버그 연동)
7. **홈/병원 탭 네비게이션 전환**
8. **알림 센터 열기/닫기**
9. **알림 읽음 처리 및 뱃지 카운트 감소**
10. **공통 토스트(Toast) 메시지 알림**

## 의도된 오류 (3개)
이 프로젝트는 강화학습 모델(PPO) 훈련을 위해 다음 3개의 오류를 포함하고 있습니다.
1. **site004-bug01 (DB 오류):** DB 날짜 비교 오류 (다가오는 일정 대신 과거 일정 노출)
2. **site004-bug02 (네트워크 오류):** 요청 폼 필수 필드 누락 검증 실패
3. **site004-bug03 (보안 오류):** 토큰 만료 등 세션 유효성 검증 누락
