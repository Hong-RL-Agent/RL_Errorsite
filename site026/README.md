# Language Buddy (site026)

## 개요
- **사이트 이름:** Language Buddy
- **포트 번호:** 9355
- **주제:** 언어 교환 매칭 서비스

## 주요 기능
- 글로벌 파트너 목록 탐색 및 원어민 필터링
- 파트너별 상세 프로필 및 학습 목표 확인
- 매칭 신청 및 실시간 상태 알림
- 매칭 요청 수락/거절 관리 기능
- 개인 프로필 및 매칭 통계 대시보드

## 의도된 오류 (3개)
1. **site026-bug01 (DB 매칭 상태 오류):** 매칭 요청을 거절하더라도 서버에서 상태를 'rejected'로 업데이트하지 않아, 목록에 계속 'pending' 상태로 남게 되는 데이터 일관성 결함이 있습니다.
2. **site026-bug02 (네트워크 요청 순서 오류):** 매칭 신청 시 응답 속도가 일정하지 않아, 짧은 시간 내에 여러 번 요청할 경우 이전 요청의 결과가 최신 상태를 덮어쓰는 레이스 컨디션이 발생할 수 있습니다.
3. **site026-bug03 (프로필 비공개 정보 노출):** 파트너 목록 API에서 일반 사용자에게 보여선 안 될 비공개 이메일과 시스템 관리용 메모 필드가 응답 JSON에 그대로 포함되어 유출되는 보안 취약점이 있습니다.
