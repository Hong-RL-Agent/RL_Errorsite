# MindLog (site028)

## 개요
- **사이트 이름:** MindLog
- **포트 번호:** 9357
- **주제:** 감정 기록 및 일기 서비스

## 주요 기능
- 오늘의 감정 아이콘 선택 및 상세 일기 작성
- 과거 날짜 선택 기능을 통한 밀린 일기 기록 가능
- 감정 키워드(태그) 추가 및 검색 기능
- 월별/주별 감정 분포 및 변화 통계 리포트
- 차분하고 감성적인 UI 기반의 사용자 경험 제공

## 의도된 오류 (3개)
1. **site028-bug01 (DB 날짜 저장 오류):** 사용자가 과거 특정 날짜를 선택하여 일기를 작성하더라도, 서버 저장 시 클라이언트의 날짜 정보를 무시하고 무조건 서버의 현재 날짜로 저장되는 결함이 있습니다.
2. **site028-bug02 (네트워크 응답 지연 처리 미흡):** 마음 통계 데이터를 불러올 때 서버의 지연 응답에 대해 프론트엔드가 적절히 대응하지 못하여, 데이터가 수신된 후에도 로딩 화면이 사라지지 않고 사용자 인터랙션을 방해하는 UX 오류가 있습니다.
3. **site028-bug03 (개인 기록 접근 제어 실패):** 일기 상세 조회 API에서 작성자 본인 여부를 검증하지 않아, `entryId`만 변경하면 타인의 지극히 사적인 감정 기록과 비밀 메모를 무단으로 열람할 수 있는 IDOR 취약점이 있습니다.
