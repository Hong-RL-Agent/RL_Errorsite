# FarmDiary (site053)

## 개요
- **사이트 이름:** FarmDiary
- **포트 번호:** 9382
- **주제:** 농작물 재배 기록 서비스

## 주요 기능
- 꿀 고구마, 방울 토마토 등 나의 작물 목록 관리 및 성장 상태 추적
- 수확량 통계 분석을 통한 연간 생산량 흐름 파악
- 작물별 재배 시작일, 현재 단계(Growing, Flowering 등) 정보 제공
- 검색 및 실시간 통계 로딩 기능을 갖춘 영농 관리 인터페이스
- 얼디 그린과 테라코타 브라운 컬러를 활용한 따뜻하고 감성적인 농장 일지 UI

## 의도된 오류 (3개)
1. **site053-bug01 (DB 수확량 합계 오류):** 수확량 통계 조회 시, 사용자가 삭제한 기록(`isDeleted: true`)까지 모두 합산에 포함되어 실제보다 과다한 수확량이 표시되는 집계 결함이 있습니다.
2. **site053-bug02 (네트워크 응답 지연):** 특정 작물(예: 방울 토마토)에 대한 통계를 조회할 때 서버 측에서 인위적인 지연을 발생시켜, 응답이 비정상적으로 늦어지는 성능 결함이 있습니다.
3. **site053-bug03 (사용자 농장 데이터 분리 실패):** 작물 목록 조회 시 로그인한 사용자의 데이터만 보여주지 않고 시스템의 모든 사용자 데이터를 반환하여, 타 농장의 재배 현황을 무단으로 열람할 수 있는 보안 취약점이 있습니다.
