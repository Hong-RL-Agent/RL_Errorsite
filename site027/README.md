# HomeFix Request (site027)

## 개요
- **사이트 이름:** HomeFix Request
- **포트 번호:** 9356
- **주제:** 집수리 요청 및 기사 매칭 서비스

## 주요 기능
- 수리 필요 분야별(전기, 배관, 인테리어) 전문 기사 탐색
- 기사별 평점 및 소개글 확인 및 매칭 요청
- 상세한 수리 요청서 작성 및 제출
- 개인별 수리 신청 내역 및 현재 진행 상태 조회
- 전문가 견적 비교 및 사용자 이용 후기 시스템

## 의도된 오류 (3개)
1. **site027-bug01 (DB 잘못된 참조 저장):** 수리 요청을 저장할 때 기사의 고유 ID(`technicianId`) 대신 해당 서비스의 카테고리 명칭이 잘못 저장되어, 내역 조회 시 기사 정보가 올바르게 표시되지 않는 데이터 참조 오류가 있습니다.
2. **site027-bug02 (네트워크 에러 본문 불일치):** 필수 입력값이 누락되어 요청에 실패(HTTP 400)했음에도 불구하고, 서버 응답 본문에는 `success: true`가 포함되어 클라이언트가 성공으로 오해하게 만드는 응답 프로토콜 결함이 있습니다.
3. **site027-bug03 (요청자 권한 검증 누락):** 특정 수리 요청의 상세 정보를 조회할 때 사용자 권한을 검증하지 않아, `requestId`만 변경하면 타인의 개인 주소나 민감한 수리 요청 내용을 무단으로 열람할 수 있는 IDOR 취약점이 있습니다.
