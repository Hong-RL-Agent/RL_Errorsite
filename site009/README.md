# GreenShop (site009)

## 개요
- **사이트 이름:** GreenShop
- **사이트 ID:** site009
- **포트 번호:** 9338
- **기술 스택:** Node.js, Express, Vanilla HTML/CSS/JS

## 실행 방법
```bash
cd site009
npm install
npm start
```
서버가 실행되면 브라우저에서 `http://localhost:9338` 으로 접속할 수 있습니다.

## API 엔드포인트 목록
- `GET /api/health` - 서버 상태 확인
- `GET /api/items` - 친환경 상품 목록 조회
- `POST /api/coupons/apply` - 쿠폰 코드 검증 및 할인액 반환
- `POST /api/orders` - 주문(결제) 생성
- `GET /api/orders` - 내 주문 내역 조회

## 정상 작동 기능 목록 (인터랙션)
1. **상품 목록 그리드 렌더링**
2. **상품명 검색 및 브랜드 필터링**
3. **가격 기준 정렬 (높은/낮은 가격순)**
4. **상품 장바구니 담기 및 알림 표시**
5. **장바구니 내 수량 조절 (+/-) 및 상품 금액 계산**
6. **쿠폰 번호 입력 및 검증/적용 (할인 금액 반영)**
7. **장바구니 총 결제 금액 동적 계산**
8. **주문 폼 제출 (POST API 연동)**
9. **주문 내역 탭 전환 및 데이터 불러오기**
10. **주문 내역 새로고침 버튼 동작**

## 의도된 오류 (3개)
이 프로젝트는 강화학습 모델(PPO) 훈련을 위해 다음 3개의 오류를 포함하고 있습니다.
1. **site009-bug01 (DB 오류):** DB 쿠폰 검증 오류 (만료된 쿠폰의 날짜 검증 누락)
2. **site009-bug02 (네트워크 오류):** 네트워크 오류 메시지 누락 (서버 실패 시 빈 응답 바디 전송)
3. **site009-bug03 (보안 오류):** 가격 조작 취약점 (클라이언트 전송 price 파라미터 무조건 신뢰)
