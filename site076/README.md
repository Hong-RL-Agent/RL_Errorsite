# TownNotice (site076)

## 개요
- **사이트 이름:** TownNotice
- **포트 번호:** 9405
- **주제:** 동네 공지 및 민원 접수 서비스

## 주요 기능
- 동네별 긴급 행정 공지 및 생활 정보 실시간 탐색 기능
- 주민들의 불편 사항을 직접 제보하고 처리 과정을 추적하는 민원 접수 시스템
- 자주 발생하는 민원에 대한 해결책을 제공하는 주민 참여형 FAQ
- 퍼블릭 그린과 소프트 그레이 컬러를 활용한 신뢰감 있고 안정적인 주민 포털 UI
- 공지사항 지역 필터링, 민원 상태 배지 시각화 등 10개 이상의 사용자 인터랙션

## 의도된 오류 (3개)
1. **site076-bug01 (DB 민원 상태 업데이트 오류):** 특정 민원의 처리 상태를 '해결됨'으로 변경하려 할 때, 서버 측 로직 결함으로 인해 요청된 민원이 아닌 목록의 첫 번째 민원 상태만 무조건 변경되는 결함이 있습니다.
2. **site076-bug02 (네트워크 요청 검증 누락):** 민원 접수 시 필수 항목인 제목(`title`)이 비어있음에도 불구하고 서버에서 유효성 검사를 수행하지 않아 제목 없는 비정상적인 민원이 성공적으로 접수되는 결함이 있습니다.
3. **site076-bug03 (민원 개인정보 접근 제어 실패):** 민원 상세 정보 조회 시 소유자 권한 확인 절차가 누락되어, `complaintId` 값 조작만으로 다른 주민의 민원 내용과 개인정보를 무단 열람할 수 있는 보안 취약점이 존재합니다.
