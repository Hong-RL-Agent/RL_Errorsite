# MarketRecipe (site074)

## 개요
- **사이트 이름:** MarketRecipe
- **포트 번호:** 9403
- **주제:** 장보기 기반 레시피 추천 서비스

## 주요 기능
- 냉장고 속 재료나 오늘 장본 식재료를 선택하여 맞춤형 레시피를 추천받는 기능
- 선택한 재료별 우리 동네 마켓 실시간 최저가 비교 테이블 제공
- 마음에 드는 레시피를 개인 영감 보드에 저장하고 관리하는 기능
- 프레시 리프 그린과 웜 머스타드 컬러를 활용한 활기찬 마켓 스타일 UI
- 재료 칩 선택, 레시피 검색, 가격 정렬 등 10개 이상의 풍부한 사용자 인터랙션

## 의도된 오류 (3개)
1. **site074-bug01 (DB 재료 필터 오류):** 여러 재료를 선택했을 때 모든 재료를 포함하는 레시피를 찾아야 하지만, 서버 측 필터링 로직 결함으로 인해 하나라도 포함되면 추천 결과에 노출되는 결함이 있습니다.
2. **site074-bug02 (네트워크 응답 Content-Type 오류):** 최저가 비교 데이터를 전송할 때 서버가 응답 헤더의 `Content-Type`을 `text/html`로 잘못 설정하여 전송함으로써, 클라이언트 측의 데이터 파싱 에러를 유발하는 결함이 있습니다.
3. **site074-bug03 (저장 레시피 권한 검증 누락):** 저장된 레시피 상세 조회 시 요청자가 소유자인지 확인하는 절차가 누락되어, `savedId` 값 조작만으로 다른 사용자의 저장 목록을 무단 열람할 수 있는 보안 취약점이 존재합니다.
