# MyCloset (site072)

## 개요
- **사이트 이름:** MyCloset
- **포트 번호:** 9401
- **주제:** 옷장 관리 및 코디 추천 서비스

## 주요 기능
- 디지털 옷장에 내 옷을 카테고리(상의, 아우터, 팬츠 등)별로 등록하고 관리하는 기능
- 보유한 아이템들을 조합하여 매일 새로운 스타일을 제안하는 코디 추천 시스템
- 계절별 필터링 및 이름/브랜드 기반의 효율적인 옷 검색 시스템
- 미니멀리스트 시크와 뉴트럴 베이지 컬러를 활용한 세련된 패션 매거진 UI
- 과거 착용 기록을 타임라인 형태로 확인하여 중복 코디를 방지하는 아카이브 기능

## 의도된 오류 (3개)
1. **site072-bug01 (DB 카테고리 매핑 오류):** 새로운 옷을 추가할 때 사용자가 선택한 카테고리(예: Outer, Pants)를 무시하고, 서버 측 로직 결함으로 인해 무조건 'Top' 카테고리로 고정되어 저장되는 결함이 있습니다.
2. **site072-bug02 (네트워크 부분 응답):** AI 코디 추천 결과 조회 시, 서버 응답 데이터에서 특정 추천 세트의 구성 아이템 일부(예: 신발 등)가 누락되어 불완전하게 표시되는 데이터 결함이 있습니다.
3. **site072-bug03 (사용자 옷장 데이터 분리 실패):** 옷 목록 API 호출 시 현재 로그인한 사용자의 데이터만 필터링하지 않고, 시스템 내 모든 사용자의 개인 옷장 데이터를 한꺼번에 반환하는 심각한 보안 데이터 노출 취약점이 존재합니다.
