# RentTool (site046)

## 개요
- **사이트 이름:** RentTool
- **포트 번호:** 9375
- **주제:** 공구 대여 서비스

## 주요 기능
- 전동 드릴, 세척기 등 다양한 산업용 및 가정용 공구 목록 조회
- 카테고리별 공구 탐색 및 대여료 정보 확인
- 원하는 날짜를 선택하여 실시간 대여 예약 기능
- 나의 대여 히스토리 및 상세 내역(주소, 상태 등) 확인
- 다크 그레이와 오렌지 컬러를 활용한 강인한 인더스트리얼 UI

## 의도된 오류 (3개)
1. **site046-bug01 (DB 대여 기간 충돌):** 이미 예약된 공구의 대여 기간과 겹치는 일정으로 새로운 예약이 신청되어도, 서버 측에서 충돌 검사를 수행하지 않아 중복 대여가 발생하는 데이터 결함이 있습니다.
2. **site046-bug02 (네트워크 요청 검증 누락):** 대여 신청 시 반납 예정일(`endDate`) 정보가 누락되어도 서버가 에러를 발생시키지 않고 성공 응답을 반환하여, 비정상적인 예약 데이터가 생성되는 결함이 있습니다.
3. **site046-bug03 (대여 내역 접근 제어 실패):** 대여 상세 조회 API에서 요청자가 실제 대여자인지 확인하는 로직이 누락되어, `rentalId` 조작을 통해 타인의 대여 정보(공구명, 배송지 등)를 무단으로 열람할 수 있는 IDOR 보안 취약점이 있습니다.
