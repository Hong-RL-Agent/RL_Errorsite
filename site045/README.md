# PlantShop (site045)

## 개요
- **사이트 이름:** PlantShop
- **포트 번호:** 9374
- **주제:** 실내 식물 쇼핑몰

## 주요 기능
- 몬스테라, 올리브 나무 등 감성적인 실내 식물 상품 목록 및 상세 조회
- 관리 난이도별(초보자용, 매니아용 등) 식물 탐색 필터 제공
- 원하는 식물을 장바구니에 담고 수량을 관리하는 기능
- 실시간 재고 확인 및 간편 주문 결제 시스템
- 나의 과거 주문 이력 조회 및 배송 상태 확인 서비스
- 보태니컬 그린 톤의 싱그럽고 편안한 식물 샵 전용 UI

## 의도된 오류 (3개)
1. **site045-bug01 (DB 재고 음수 허용):** 주문 시 상품의 현재 재고 수량을 검증하지 않아, 보유 재고보다 많은 수량을 주문할 경우 DB상의 재고 수량이 음수(`-9` 등)로 저장되는 결함이 있습니다.
2. **site045-bug02 (네트워크 Content-Type 오류):** 주문 성공 시 서버가 JSON 응답을 보내면서 헤더의 `Content-Type`을 `text/plain`으로 잘못 설정하여, 클라이언트에서 응답 데이터를 자동으로 처리하지 못하는 네트워크 결함이 있습니다.
3. **site045-bug03 (주문 금액 변조):** 서버 측에서 결제 총액을 재계산하여 검증하지 않고 클라이언트가 전송한 `discountedTotal` 파라미터를 그대로 최종 결제액으로 채택하여, 사용자가 임의로 가격을 조작할 수 있는 보안 취약점이 있습니다.
