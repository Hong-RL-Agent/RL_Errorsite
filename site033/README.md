# OfficeSupply Mall (site033)

## 개요
- **사이트 이름:** OfficeSupply Mall
- **포트 번호:** 9362
- **주제:** 사무용품 주문 서비스

## 주요 기능
- 기업용 사무용품 카테고리별 탐색 및 검색
- 장바구니 담기, 수량 조절 및 실시간 금액 합계 확인
- 대량 구매를 위한 공식 견적서 발행 및 관리 기능
- 상품 상세 정보 확인 및 재고 현황 시뮬레이션
- 주문 프로세스 및 결제 완료 후 내역 확인

## 의도된 오류 (3개)
1. **site033-bug01 (DB 주문 합계 계산 오류):** 주문을 처리할 때 서버 측에서 각 품목의 수량을 곱하지 않고 단가만 합산하여 최종 주문 금액(`totalAmount`)이 비정상적으로 낮게 계산되는 결함이 있습니다.
2. **site033-bug02 (네트워크 실패 응답 누락):** 특정 조건(예: 과도한 수량 주문)에서 주문이 실패할 경우, 서버가 실패 원인이 담긴 JSON 대신 본문이 비어 있는 `500 Internal Server Error`만 반환하여 클라이언트가 원인을 파악할 수 없는 결함이 있습니다.
3. **site033-bug03 (견적 금액 조작 취약점):** 공식 견적서를 발행할 때 서버가 견적 총액을 재계산하지 않고 클라이언트가 보낸 `quoteTotal` 값을 그대로 저장하여, 요청 패킷 조작을 통해 견적 금액을 마음대로 변경할 수 있는 보안 취약점이 있습니다.
