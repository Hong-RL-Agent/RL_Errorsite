# Recipe Cloud (site017)

## 개요
- **사이트 이름:** Recipe Cloud
- **포트 번호:** 9346
- **주제:** 레시피 저장 및 추천 서비스

## 주요 기능
- 재료 키워드 기반 레시피 검색 및 목록 조회
- 레시피 상세 정보 확인 (재료, 난이도, 시간 등)
- 마음에 드는 레시피 저장(Bookmark) 기능
- 요리 후기 작성 및 커뮤니티 피드 조회
- 레시피 난이도별 배지 표시

## 의도된 오류 (3개)
1. **site017-bug01 (DB LIKE 검색 오류):** 재료 검색 시 부분 일치(`LIKE` 검색)가 작동하지 않고, 정확히 일치하는 단어를 입력했을 때만 결과가 반환되는 검색 로직의 결함이 있습니다.
2. **site017-bug02 (잘못된 응답 코드):** 레시피 저장 시 이미 저장된 항목이라 저장이 실패한 경우에도 서버가 항상 `201 Created` 성공 코드를 반환하여 클라이언트에 혼란을 줍니다.
3. **site017-bug03 (입력값 검증 누락):** 후기 작성 시 서버 측에서 입력값 검증 및 필터링을 수행하지 않아, 악의적인 스크립트(XSS)나 무제한 길이의 텍스트가 DB에 그대로 저장될 수 있는 보안 취약점이 있습니다.
