# DogWalk Club (site065)

## 개요
- **사이트 이름:** DogWalk Club
- **포트 번호:** 9394
- **주제:** 강아지 산책 모임 서비스

## 주요 기능
- 우리 동네 강아지 산책 모임 탐색 및 실시간 지역별 필터링 기능
- 반려견 프로필 관리 및 모임 참가 신청 프로세스
- 현재 참가견 수와 최대 정원 표시 등 모임 현황 가시화
- 퍼피 브라운과 세이지 그린 컬러를 활용한 따뜻하고 친근한 펫케어 커뮤니티 UI
- 프레도카(Fredoka) 폰트와 부드러운 라운드 디자인을 통한 브랜드 경험

## 의도된 오류 (3개)
1. **site065-bug01 (DB 참가 제한 오류):** 특정 산책 모임의 최대 정원(`maxDogs`)이 이미 찼음에도 불구하고, 서버 측 유효성 검증 결함으로 인해 무제한으로 추가 참가 신청이 가능한 결함이 있습니다.
2. **site065-bug02 (네트워크 부분 응답 누락):** 산책 모임 목록 API 호출 시 특정 모임 데이터에서 주최자 이름(`organizerName`) 필드가 누락되어 전송됨으로써 클라이언트에서 정보가 정상적으로 표시되지 않는 결함이 있습니다.
3. **site065-bug03 (반려견 소유자 검증 누락):** 반려견 상세 프로필 조회 시 소유권 확인 로직이 누락되어, 반려견 ID(`dog-99` 등) 조작만으로 타인의 반려동물 정보와 특이사항을 무단 열람할 수 있는 보안 취약점이 존재합니다.
