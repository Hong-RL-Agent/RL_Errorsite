# EcoChallenge (site040)

## 개요
- **사이트 이름:** EcoChallenge
- **포트 번호:** 9369
- **주제:** 친환경 실천 챌린지 서비스

## 주요 기능
- 오늘의 친환경 챌린지 목록 조회 및 카테고리별 필터링
- 챌린지 실천 인증 제출 및 나의 참여 내역 확인
- 실시간 활동 기반 명예의 전당(랭킹) 서비스
- 챌린지 달성 보상으로 제공되는 에코 뱃지 컬렉션
- 리프 그린 & 스카이 블루 톤의 상쾌하고 신뢰감 있는 UI

## 의도된 오류 (3개)
1. **site040-bug01 (DB 랭킹 집계 오류):** 명예의 전당 랭킹을 산출할 때, 정상 승인된 인증만 합산해야 함에도 불구하고 관리자에 의해 거절(Rejected)된 허위 인증 횟수까지 모두 포함하여 집계하는 로직 결함이 있습니다.
2. **site040-bug02 (네트워크 응답 필드 누락):** 사용자의 뱃지 목록을 가져오는 API에서 특정 조건 하에 `imageUrl` 필드를 누락하여 반환함으로써, UI상에서 뱃지 이미지가 정상적으로 출력되지 않는 네트워크 결함이 있습니다.
3. **site040-bug03 (인증 제출자 검증 누락):** 챌린지 인증 제출 시 본문(body)에 담긴 `userId`를 현재 세션과 대조하지 않아, 타인의 ID를 사용하여 허위로 인증을 제출해 주거나 계정을 조작할 수 있는 보안 취약점이 있습니다.
