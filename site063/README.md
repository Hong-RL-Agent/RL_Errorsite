# WeddingPlan Mini (site063)

## 개요
- **사이트 이름:** WeddingPlan Mini
- **포트 번호:** 9392
- **주제:** 웨딩 준비 체크리스트 서비스

## 주요 기능
- 예식장, 스튜디오 등 결혼 준비 단계를 체계적으로 관리하는 체크리스트 기능
- 총 예상 지출액과 항목별 예산을 한눈에 파악하는 대시보드 및 지출 추적
- 소중한 분들을 초대하기 위한 하객 명단 작성 및 관계별 분류 관리
- 클래식 화이트와 로즈 쿼츠 컬러를 활용한 우아하고 로맨틱한 웨딩 플래너 UI
- 플레이페어 디스플레이(Playfair Display) 폰트를 통한 고급스러운 타이포그래피 경험

## 의도된 오류 (3개)
1. **site063-bug01 (DB 예산 합계 오류):** 예산 요약 대시보드에서 지출액을 계산할 때, 취소 처리된 항목(`isCancelled: true`)을 제외하지 않고 모두 합산하여 실제 지출보다 높게 표시되는 계산 결함이 있습니다.
2. **site063-bug02 (네트워크 요청 본문 검증 실패):** 하객 명단 추가 API 호출 시 필수값인 하객 이름(`name`)이 비어 있어도 서버에서 유효성 검증을 수행하지 않고 성공적으로 데이터를 저장하는 정책 결함이 있습니다.
3. **site063-bug03 (공유 일정 권한 검증 누락):** 타 커플의 준비 일정 목록을 조회할 때 소유권 확인 절차가 누락되어, 플랜 ID(`plan-999` 등) 조작만으로 타인의 프라이빗한 웨딩 준비 일정을 무단 열람할 수 있는 보안 취약점이 존재합니다.
