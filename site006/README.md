# MealBox Order (site006)

## 개요
- **사이트 이름:** MealBox Order
- **사이트 ID:** site006
- **포트 번호:** 9335
- **기술 스택:** Node.js, Express, Vanilla HTML/CSS/JS

## 실행 방법
```bash
cd site006
npm install
npm start
```
서버가 실행되면 브라우저에서 `http://localhost:9335` 로 접속할 수 있습니다.

## API 엔드포인트 목록
- `GET /api/health` - 서버 상태 확인
- `GET /api/menus` - 도시락 메뉴 목록 조회 (재고 포함)
- `POST /api/orders` - 주문 생성
- `GET /api/orders` - 주문 내역(배송 현황) 조회
- `GET /api/reviews` - 리뷰 목록 조회
- `POST /api/reviews` - 리뷰 작성

## 정상 작동 기능 목록 (인터랙션)
1. **메뉴 목록 및 재고 상태 조회/렌더링**
2. **메뉴 이름 텍스트 검색 및 카테고리 필터링**
3. **메뉴 상세 정보 모달 팝업 열기/닫기**
4. **장바구니 담기 및 장바구니 UI 업데이트**
5. **장바구니 아이템 수량 조절 (+/-)**
6. **장바구니 총 금액 실시간 계산**
7. **주문(결제) 폼 제출 (POST API 연동)**
8. **주문 내역(배송 현황) 조회 및 새로고침**
9. **리뷰 조회 및 새 리뷰 작성 폼 제출**
10. **도시락 주문 / 배송 현황 탭 네비게이션 전환**

## 의도된 오류 (3개)
이 프로젝트는 강화학습 모델(PPO) 훈련을 위해 다음 3개의 오류를 포함하고 있습니다.
1. **site006-bug01 (DB 오류):** DB 재고 차감 오류 (재고 부족해도 결제 성공 및 마이너스 차감)
2. **site006-bug02 (네트워크 오류):** 네트워크 재시도 중복 요청 (동일 주문 반복 생성 허용)
3. **site006-bug03 (보안 오류):** 결제 금액 조작 취약점 (클라이언트 전송 총액 무조건 신뢰)
