# MoviePick (site008)

## 개요
- **사이트 이름:** MoviePick
- **사이트 ID:** site008
- **포트 번호:** 9337
- **기술 스택:** Node.js, Express, Vanilla HTML/CSS/JS

## 실행 방법
```bash
cd site008
npm install
npm start
```
서버가 실행되면 브라우저에서 `http://localhost:9337` 으로 접속할 수 있습니다.

## API 엔드포인트 목록
- `GET /api/health` - 서버 상태 확인
- `GET /api/movies` - 영화 목록 조회 (평점 집계 포함)
- `GET /api/reviews` - 영화 리뷰 목록 조회
- `POST /api/reviews` - 새로운 리뷰 등록
- `GET /api/favorites` - 나의 찜 목록 조회
- `POST /api/favorites` - 찜하기 추가/삭제 토글
- `GET /api/preferences` - 나의 취향 분석 태그 조회
- `POST /api/preferences` - 취향 분석 태그 저장

## 정상 작동 기능 목록 (인터랙션)
1. **영화 목록 및 포스터 렌더링 (그리드 뷰)**
2. **영화 제목 텍스트 검색 및 장르 필터링, 정렬(최신순/평점순)**
3. **영화 상세 정보 모달 팝업 열기/닫기**
4. **리뷰 목록 렌더링 및 리뷰 폼을 통한 작성 연동**
5. **영화 찜하기 하트 토글 (모달 내부 및 그리드 카드 내부)**
6. **실시간 인기 리뷰 가로 스크롤(캐러셀) 뷰 렌더링**
7. **홈 화면 상단 추천 영화 강제 새로고침 기능**
8. **취향 분석 태그(해시태그) 다중 선택 토글 및 저장**
9. **네비게이션 탭 전환 (홈 / 장르별 / 찜 목록 / 취향 분석)**
10. **커스텀 알림(토스트) 팝업 제공**

## 의도된 오류 (3개)
이 프로젝트는 강화학습 모델(PPO) 훈련을 위해 다음 3개의 오류를 포함하고 있습니다.
1. **site008-bug01 (DB 오류):** DB 집계 오류 (영화 평균 평점 계산 시 고정값 사용)
2. **site008-bug02 (네트워크 오류):** 잘못된 Content-Type 응답 (JSON 데이터에 `text/plain` 명시)
3. **site008-bug03 (보안 오류):** 저장형 XSS 취약점 (리뷰 내용 내 HTML 태그 미처리)

자세한 오류 정보는 `BUGS.md`를 확인하세요.
작업 진행 상태는 `TODO.md`를 참고하세요.
