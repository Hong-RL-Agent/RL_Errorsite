# ArtClass Studio (site013)

## 개요
- **사이트 이름:** ArtClass Studio
- **포트 번호:** 9342
- **주제:** 온라인 미술 클래스 예약 서비스

## 주요 기능
- 미술 클래스 목록 조회 및 난이도별 필터링
- 클래스 상세 정보 확인 및 예약 신청 (날짜, 이름)
- 수강 후기 조회 및 삭제 (관리자 기능 시뮬레이션)
- 내 예약 현황 확인 및 관리
- 강사 소개 및 작품 갤러리 뷰

## 의도된 오류 (3개)
1. **site013-bug01 (DB 외래키 검증 누락):** 예약 생성 시 서버가 `classId`의 유효성을 검사하지 않아, 존재하지 않는 클래스에 대해서도 예약이 생성되어 데이터 무결성이 파괴됩니다.
2. **site013-bug02 (HTTP 메서드 오용):** 후기 삭제 기능을 `DELETE`가 아닌 `GET` 메서드로 구현하여, 링크 클릭만으로 데이터가 변경되는 보안 및 설계적 결함이 있습니다.
3. **site013-bug03 (CSRF 방어 누락):** 예약 생성 API에 CSRF 방어 기제가 없어, 교차 사이트 요청 위조 공격을 통해 사용자의 의사와 무관하게 예약을 생성할 수 있습니다.
