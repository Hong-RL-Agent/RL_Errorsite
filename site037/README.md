# PhotoPrint Lab (site037)

## 개요
- **사이트 이름:** PhotoPrint Lab
- **포트 번호:** 9366
- **주제:** 사진 인화 주문 서비스

## 주요 기능
- 인화 옵션(4x6, 5x7, 8x10 등) 선택 및 실시간 단가 확인
- 사진 업로드 시뮬레이션 및 장바구니 관리
- 배송지 입력 및 최종 주문 결제 프로세스
- 주문 내역 조회를 통한 과거 인화 내역 및 배송 상태 확인
- 사진별 수량 조절 및 삭제 등 편리한 주문 인터랙션

## 의도된 오류 (3개)
1. **site037-bug01 (DB 옵션 가격 매핑 오류):** 주문 시 서버에서 인화 옵션을 식별하는 인덱스 계산 오류가 있어, 사용자가 선택한 크기와 실제 적용되는 가격이 일치하지 않는 결함이 있습니다.
2. **site037-bug02 (네트워크 큰 요청 처리 실패):** 주문 데이터의 용량이 서버 제한(1MB)을 초과할 경우, 표준 에러인 `413 Payload Too Large`를 반환하지 않고 일반적인 `500 Internal Server Error`를 반환하여 오류 원인을 파악하기 어렵게 만드는 결함이 있습니다.
3. **site037-bug03 (배송지 정보 과다 노출):** 주문 내역 조회 API가 목록 단계에서도 사용자의 상세 주소와 관리용 배송 메모(공동현관 비밀번호 등 민감 정보 포함)를 불필요하게 모두 반환하는 개인정보 노출 취약점이 있습니다.
