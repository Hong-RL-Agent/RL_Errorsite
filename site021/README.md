# SkillMap Academy (site021)

## 개요
- **사이트 이름:** SkillMap Academy
- **포트 번호:** 9350
- **주제:** 온라인 강의 수강 관리 서비스

## 주요 기능
- 카테고리별 강의 탐색 및 검색
- 강의 상세 정보 조회 및 수강생 전용 콘텐츠 보기
- 수강 중인 강의의 진도율 대시보드 확인
- 데일리 퀴즈 풀기 및 실시간 자동 채점
- 마이페이지를 통한 포인트 및 프로필 관리

## 의도된 오류 (3개)
1. **site021-bug01 (DB 진도율 계산 오류):** 진도율(%)을 계산할 때 분모인 '전체 강의 수'를 실제 데이터가 아닌 고정값(10)으로 나누어, 강의 수에 따라 진도율이 100%를 넘거나 실제와 다르게 표시되는 논리적 오류가 있습니다.
2. **site021-bug02 (네트워크 부분 실패):** 3문항의 퀴즈를 제출했을 때, 서버가 의도적으로 앞의 2문항에 대한 결과만 반환하고 마지막 문항의 채점 결과를 누락시켜 클라이언트에 응답합니다.
3. **site021-bug03 (비인가 강의 접근):** 수강 신청을 하지 않은 강의임에도 불구하고, 강의 ID를 이용해 직접 API를 호출하면 서버가 권한을 검증하지 않고 유료 강의 콘텐츠를 반환하는 보안 취약점이 있습니다.
