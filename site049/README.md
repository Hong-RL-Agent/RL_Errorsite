# TodoSprint (site049)

## 개요
- **사이트 이름:** TodoSprint
- **포트 번호:** 9378
- **주제:** 팀 할 일 관리 서비스

## 주요 기능
- 프로젝트별 할 일 목록 관리 및 담당자 지정 기능
- Todo, Doing, Done의 3단계 상태 전환을 통한 칸반 보드 시스템
- 할 일 검색 및 프로젝트 필터링을 통한 실시간 업무 추적
- 직관적인 모달 인터페이스를 통한 작업 상세 내용 확인 및 업데이트
- 프로스트 블루 톤의 깔끔하고 현대적인 SaaS 협업 툴 UI

## 의도된 오류 (3개)
1. **site049-bug01 (DB 상태 업데이트 대상 오류):** 할 일의 상태를 업데이트할 때 고유 ID(`taskId`)가 아닌 단순 배열 인덱스를 기준으로 데이터를 수정하여, 요청한 작업이 아닌 전혀 다른 작업의 상태가 변경되는 데이터 무결성 결함이 있습니다.
2. **site049-bug02 (네트워크 경쟁 상태):** 상태 변경 요청 시 서버의 응답 지연을 무작위로 설정하여, 빠른 연속 클릭 시 이전 요청의 응답이 최신 로컬 상태를 덮어써버려 상태가 되돌아가는 현상이 발생하는 결함이 있습니다.
3. **site049-bug03 (프로젝트 권한 검증 누락):** 특정 프로젝트의 할 일을 조회할 때 요청자가 해당 팀원인지 검증하지 않아, `projectId` 조작을 통해 타 팀의 비공개 프로젝트 업무 리스트를 무단 조회할 수 있는 보안 취약점이 존재합니다.
