# Freelance Board (site066)

## 개요
- **사이트 이름:** Freelance Board
- **포트 번호:** 9395
- **주제:** 프리랜서 프로젝트 매칭 서비스

## 주요 기능
- 웹 사이트 구축, 앱 디자인, 데이터 분석 등 다양한 아웃소싱 프로젝트 탐색
- React, Node.js, Python 등 요구되는 기술 스택별 실시간 필터링 및 검색
- 프로젝트 상세 정보 확인 및 맞춤형 제안서(금액, 메시지) 제출 프로세스
- 프로페셔널 인디고와 클린 그래파이트 컬러를 활용한 신뢰감 있는 비즈니스 UI
- 내 지원 현황 관리 및 클라이언트 평점/후기 확인 기능

## 의도된 오류 (3개)
1. **site066-bug01 (DB 제안 중복 검증 누락):** 특정 프로젝트에 이미 제안서를 제출했음에도 불구하고, 서버 측 중복 검증 로직 결함으로 인해 동일한 사용자가 여러 번 중복하여 지원할 수 있는 결함이 있습니다.
2. **site066-bug02 (네트워크 상태 코드 오류):** 모집이 이미 마감된 프로젝트에 제안서를 제출하려고 할 때, 비즈니스 로직상으로는 실패했음에도 서버가 HTTP 상태 코드를 `201 Created`로 반환하여 클라이언트에 성공한 것처럼 오인하게 만드는 정책 결함이 있습니다.
3. **site066-bug03 (클라이언트 전용 정보 노출):** 프로젝트 목록 및 상세 정보를 불러올 때, 일반 프리랜서에게는 노출되어서는 안 되는 클라이언트의 관리용 내부 메모(`clientInternalMemo`)와 비공개 연락처가 필터링 없이 노출되는 보안 취약점이 존재합니다.
