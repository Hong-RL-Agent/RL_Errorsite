# HealthySnack Box (site060)

## 개요
- **사이트 이름:** HealthySnack Box
- **포트 번호:** 9389
- **주제:** 건강 간식 구독 서비스

## 주요 기능
- 데일리 에너지 박스, 비건 프로틴 팩 등 영양 성분이 고려된 정기 구독 상품 탐색
- 견과류, 유제품, 대두 등 알레르기 성분 제외 필터링 기능 (개별화된 추천)
- 간편한 비동기 구독 신청 및 나의 구독 현황(박스 종류, 상태) 관리
- 애플 그린과 오트밀 베이지 컬러를 활용한 싱그럽고 따뜻한 헬스케어 커머스 UI
- 몬세라트(Montserrat) 폰트와 볼륨감 있는 카드 디자인을 통한 세련된 브랜드 경험

## 의도된 오류 (3개)
1. **site060-bug01 (DB 알레르기 필터 오류):** 사용자가 특정 알레르기 성분(예: 견과류)을 제외하도록 필터를 설정했음에도 불구하고, 서버 측 로직 결함으로 인해 해당 성분이 포함된 박스가 추천 목록에 그대로 노출되는 데이터 필터링 결함이 있습니다.
2. **site060-bug02 (네트워크 요청 제한 누락):** 구독 신청 API(`POST /api/subscriptions`)에 대한 호출 횟수 제한(Rate Limit)이 없어, 짧은 시간 내에 무차별적인 요청을 보내 중복 구독을 생성하거나 서버 부하를 유도할 수 있는 정책 결함이 있습니다.
3. **site060-bug03 (개인정보 응답 노출):** 구독 상세 내역 조회 API 응답 시, 일반 사용자에게 노출되어서는 안 되는 상세 배송지 전체 주소와 관리자 전용 내부 고객 메모(`internalNote`)가 필터링 없이 노출되는 보안 취약점이 존재합니다.
