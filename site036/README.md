# Campus Club (site036)

## 개요
- **사이트 이름:** Campus Club
- **포트 번호:** 9365
- **주제:** 대학 동아리 소개 및 가입 신청 서비스

## 주요 기능
- 카테고리별(학술, 예술, 운동 등) 동아리 목록 조회 및 검색
- 동아리별 상세 정보(소개, 지원 자격) 및 모집 상태 확인
- 온라인 동아리 가입 신청 및 본인의 신청 현황(대기, 취소 등) 관리
- 캠퍼스 내 주요 공지사항 게시판 운영
- 관리자 권한을 통한 전체 공지사항 등록 기능

## 의도된 오류 (3개)
1. **site036-bug01 (DB 모집 상태 오류):** 동아리 정보를 반환할 때 이미 모집이 마감된 동아리임에도 불구하고 무조건 'Recruiting(모집 중)' 상태로 데이터를 조작하여 사용자에게 오도된 정보를 제공하는 결함이 있습니다.
2. **site036-bug02 (잘못된 HTTP 메서드 허용):** 가입 신청을 취소하는 중요한 상태 변경 작업을 수행할 때, 보안상 취약한 GET 메서드를 통해서도 요청이 처리되도록 설계된 네트워크 구조적 결함이 있습니다.
3. **site036-bug03 (관리자 공지 권한 검증 누락):** 전체 공지사항을 등록하는 관리자 전용 API에서 호출자의 권한을 검증하지 않아, 일반 사용자 계정으로도 무단으로 공지를 작성할 수 있는 보안 인가 취약점이 있습니다.
