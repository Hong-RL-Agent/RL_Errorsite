# Campus LostFound (site064)

## 개요
- **사이트 이름:** Campus LostFound
- **포트 번호:** 9393
- **주제:** 대학 분실물 찾기 서비스

## 주요 기능
- 캠퍼스 내 분실물 및 습득물 게시글 탐색 및 실시간 검색 기능
- 장소(도서관, 학생회관 등) 및 물품 상태(Lost/Found)별 필터링 시스템
- 습득물 신고 및 분실물 등록을 위한 간편한 게시글 작성 폼 제공
- 스카이 블루와 캠퍼스 옐로우 컬러를 활용한 밝고 활기찬 대학 커뮤니티 UI
- 게시글 상태 변경(해결됨 등)을 통한 유연한 분실물 추적 관리

## 의도된 오류 (3개)
1. **site064-bug01 (DB 상태 필터 오류):** 메인 목록에서 '습득 완료(Found)' 상태의 게시글만 보기 위해 필터를 적용해도, 서버 로직 결함으로 인해 '분실 신고(Lost)' 항목들이 함께 반환되는 필터링 결함이 있습니다.
2. **site064-bug02 (네트워크 오래된 캐시):** 새로운 게시글을 성공적으로 등록한 직후 목록으로 돌아가면, 서버에서 즉시 갱신되지 않은 이전 캐시 데이터를 반환하여 방금 등록한 글이 일시적으로 보이지 않는 정합성 결함이 있습니다.
3. **site064-bug03 (게시글 소유자 검증 누락):** 게시글의 상태를 '해결됨' 등으로 변경하는 API 요청 시 작성자 본인 여부를 확인하지 않아, 게시글 ID 조작만으로 타인의 게시글 상태를 임의로 변경할 수 있는 보안 권한 취약점이 존재합니다.
