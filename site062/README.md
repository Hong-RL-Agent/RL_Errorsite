# DevSnippet Vault (site062)

## 개요
- **사이트 이름:** DevSnippet Vault
- **포트 번호:** 9391
- **주제:** 코드 스니펫 저장 서비스

## 주요 기능
- JavaScript, TypeScript, Python 등 언어별 유용한 코드 스니펫 저장 및 탐색
- 태그 기반의 효율적인 분류 시스템과 실시간 검색 필터
- 코드 복사 버튼을 통한 즉각적인 클립보드 활용 및 사용 편의성 제공
- 나만 볼 수 있는 비공개 스니펫 설정 및 관리 기능
- 다크 코딩 테마와 일렉트릭 퍼플 컬러를 활용한 세련된 개발자 도구 UI

## 의도된 오류 (3개)
1. **site062-bug01 (DB 태그 관계 오류):** 새로운 스니펫을 저장할 때 사용자가 선택한 여러 개의 태그 중 첫 번째 태그만 데이터베이스 관계에 연결되어 저장되는 데이터 정합성 결함이 있습니다.
2. **site062-bug02 (네트워크 응답 포맷 오류):** 태그 목록 조회 API 호출 시, 서버가 JSON 배열이 아닌 쉼표로 구분된 단일 문자열 형태의 응답을 반환하여 클라이언트 파싱에 혼선을 주는 결함이 있습니다.
3. **site062-bug03 (비공개 스니펫 접근 제어 실패):** 작성자가 비공개(`isPrivate: true`)로 설정한 스니펫이라도, 해당 스니펫의 고유 ID를 알고 직접 API로 요청할 경우 소유권 확인 없이 내용을 열람할 수 있는 보안 취약점이 존재합니다.
