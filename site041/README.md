# BabyCare Log (site041)

## 개요
- **사이트 이름:** BabyCare Log
- **포트 번호:** 9370
- **주제:** 육아 기록 관리 서비스

## 주요 기능
- 수유(분유, 모유, 이유식) 시간 및 용량 실시간 기록 및 조회
- 아기의 몸무게, 키 등 주요 성장 지표 추적 관리
- 수면 시간 및 패턴 기록을 통한 아이의 컨디션 파악
- 보호자(엄마, 아빠, 할머니 등) 간 육아 데이터 공유 및 공동 관리
- 부드러운 파스텔 톤의 따뜻하고 아늑한 육아 전용 UI/UX

## 의도된 오류 (3개)
1. **site041-bug01 (DB 날짜 범위 조회 오류):** 수유 기록 조회 시 사용자가 선택한 특정 날짜의 데이터만 가져와야 하지만, 필터링 로직 결함으로 인해 전체 기간의 모든 수유 기록을 반환하는 오류가 있습니다.
2. **site041-bug02 (네트워크 스키마 불일치):** 성장 기록 API에서 아기의 키 정보를 전송할 때, 프론트엔드가 기대하는 `heightCm` 대신 `height`라는 잘못된 필드명을 사용하여 UI에 데이터가 표시되지 않는 결함이 있습니다.
3. **site041-bug03 (보호자 권한 검증 누락):** 공유 기록 조회 API에서 요청자가 실제 해당 아이의 보호자인지 검증하지 않아, `guardianId` 조작을 통해 타인의 육아 일지와 민감한 건강 데이터를 무단으로 열람할 수 있는 보안 취약점이 있습니다.
