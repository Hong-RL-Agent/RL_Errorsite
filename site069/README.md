# BookClub Meet (site069)

## 개요
- **사이트 이름:** BookClub Meet
- **포트 번호:** 9398
- **주제:** 독서 모임 일정 관리 서비스

## 주요 기능
- 고전 문학부터 현대 소설까지 다양한 독서 모임 일정 탐색 및 참가 신청
- 모임별 토론 발제문(질문) 확인 및 깊이 있는 독서 나눔 지원
- 신청한 모임 목록 관리 및 참여 상태 실시간 확인
- 앤틱 베이지와 딥 오크 컬러를 활용한 지적이고 클래식한 서재 분위기 UI
- 노토 세리프(Noto Serif) 폰트를 통한 가독성 높은 텍스트 환경 제공

## 의도된 오류 (3개)
1. **site069-bug01 (DB 참가 상태 오류):** 사용자가 모임 참가를 취소했을 때, 서버 측에서 성공 응답을 보내지만 실제 데이터베이스의 참가 상태(`status`)를 `active`로 유지하여 목록에서 사라지지 않는 결함이 있습니다.
2. **site069-bug02 (네트워크 응답 필드 누락):** 특정 모임의 토론 질문 목록을 불러올 때, 일부 질문 데이터에서 고유 식별자인 `questionId` 필드가 누락되어 전송되는 데이터 정합성 결함이 있습니다.
3. **site069-bug03 (비공개 모임 접근 실패):** 비공개(`isPrivate: true`)로 설정된 특정 모임의 상세 정보 요청 시, 권한 확인 절차가 누락되어 모임 ID 조작만으로 프라이빗한 모임 정보를 무단 열람할 수 있는 보안 취약점이 존재합니다.
