# RunCrew (site043)

## 개요
- **사이트 이름:** RunCrew
- **포트 번호:** 9372
- **주제:** 러닝 모임 참여 서비스

## 주요 기능
- 전국 각지의 러닝 모임 일정 조회 및 코스 난이도/거리별 필터링
- 원클릭 참가 신청 기능을 통한 커뮤니티 활동 참여
- 개인별 러닝 기록(거리, 시간, 페이스) 저장 및 수정 관리
- 크루 내 활동량을 기반으로 한 실시간 활동 랭킹 보드
- 일렉트릭 옐로우 & 블랙 테마의 역동적이고 스포티한 다크 모드 UI

## 의도된 오류 (3개)
1. **site043-bug01 (DB 참가 인원 초과):** 러닝 모임의 최대 정원(`maxParticipants`)이 초과되었음에도 서버 측 유효성 검사 미흡으로 인해 계속해서 참가 신청이 받아들여지는 데이터 정합성 결함이 있습니다.
2. **site043-bug02 (네트워크 중복 응답 처리):** 참가 신청 시 동일 사용자의 중복 신청 여부를 확인하지 않아, 네트워크 재시도나 버튼 연타 시 한 명의 사용자가 동일 모임에 여러 번 중복 등록되는 로직 결함이 있습니다.
3. **site043-bug03 (기록 소유자 검증 누락):** 러닝 기록 수정 API에서 요청자가 실제 기록의 소유자인지 대조하는 로직이 누락되어, `recordId` 조작을 통해 타인의 러닝 데이터를 무단으로 수정할 수 있는 보안 취약점이 있습니다.
