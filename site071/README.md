# LocalClass Finder (site071)

## 개요
- **사이트 이름:** LocalClass Finder
- **포트 번호:** 9400
- **주제:** 지역 원데이 클래스 탐색 서비스

## 주요 기능
- DIY, 미술, 요리 등 다양한 카테고리의 원데이 클래스 실시간 탐색
- 성수, 홍대, 수원 등 지역별 필터링 및 날짜 기반 검색 기능
- 클래스 상세 정보(가격, 위치, 강사 약력) 확인 및 비동기 예약 시스템
- 바이브런트 코랄과 소프트 샌드 컬러를 활용한 따뜻하고 세련된 라이프스타일 UI
- 내 예약 현황 관리 및 강사별 전문 프로필 조회 인터페이스

## 의도된 오류 (3개)
1. **site071-bug01 (DB 예약 정원 검증 누락):** 클래스의 수강 정원이 이미 가득 찼음에도 불구하고, 서버 측 유효성 검사 로직 결함으로 인해 무제한으로 추가 예약이 수락되는 정원 관리 결함이 있습니다.
2. **site071-bug02 (API 스키마 불일치):** 클래스 목록 조회 API 응답 시, 클라이언트가 잔여석 표시를 위해 기대하는 필드명(`seatsLeft`) 대신 `remainSeats`라는 필드명을 사용하여 데이터 렌더링에 실패하는 스키마 불일치 결함이 있습니다.
3. **site071-bug03 (예약 인증 검증 누락):** 회원 전용 기능인 클래스 예약 시, 서버 측에서 사용자의 인증 토큰 존재 여부를 검증하지 않아 로그인하지 않은 사용자도 API 직접 호출을 통해 예약을 성공시킬 수 있는 보안 취약점이 존재합니다.
