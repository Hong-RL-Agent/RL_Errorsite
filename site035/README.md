# Concert Radar (site035)

## 개요
- **사이트 이름:** Concert Radar
- **포트 번호:** 9364
- **주제:** 콘서트 일정 및 티켓 알림 서비스

## 주요 기능
- 현재 인기 있는 콘서트 목록 조회 및 아티스트/장르 필터링
- 티켓 오픈 전 실시간 알림(Email, SMS) 신청 기능
- 개별 공연의 실시간 좌석 현황(잔여 좌석 수) 조회 시뮬레이션
- 관심 공연 등록 및 나만의 알림 설정 관리
- 공연장 위치 및 주요 시설 정보 제공

## 의도된 오류 (3개)
1. **site035-bug01 (DB 알림 중복 저장):** 동일한 공연에 대해 중복 검사 로직이 없어, 사용자가 같은 공연에 대한 알림을 여러 번 반복해서 등록할 수 있는 데이터 결함이 있습니다.
2. **site035-bug02 (캐시 제어 오류):** 좌석 잔여 상태 조회 API에서 서버가 만료 시간이 긴 캐시 데이터를 반환하도록 설정되어 있어, 실제 실시간 현황이 아닌 오래된 데이터를 사용자에게 보여주는 결함이 있습니다.
3. **site035-bug03 (알림 설정 소유자 검증 누락):** 특정 알림 설정 상세 조회 시 요청자의 권한을 검증하지 않아, `alertId` 조작을 통해 타인의 알림 설정 정보(관심 공연, 연락처 수단 등)를 무단으로 열람할 수 있는 IDOR 보안 취약점이 있습니다.
