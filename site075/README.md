# MentorCircle (site075)

## 개요
- **사이트 이름:** MentorCircle
- **포트 번호:** 9404
- **주제:** 멘토링 신청 및 일정 관리 서비스

## 주요 기능
- IT, 디자인, 마케팅 등 다양한 분야의 현직 전문가(멘토) 실시간 탐색 및 프로필 조회
- 멘토별 상담 가능 시간대 확인 및 간편한 온라인 상담 신청 시스템
- 신청한 멘토링의 진행 상태(대기, 승인, 완료)를 한눈에 관리하는 개인화 보드
- 프로페셔널 네이비와 스카이 블루 컬러를 활용한 신뢰감 있고 정돈된 비즈니스 UI
- 멘토 검색, 분야별 필터링, 상담 내용 작성 등 10개 이상의 사용자 인터랙션

## 의도된 오류 (3개)
1. **site075-bug01 (DB 시간대 중복 신청):** 동일한 멘토의 동일한 상담 시간대에 이미 승인된 신청이 있더라도, 서버 측 중복 검증 로직 결함으로 인해 여러 명의 사용자가 동시에 같은 시간을 예약할 수 있는 결함이 있습니다.
2. **site075-bug02 (네트워크 오류 본문 불일치):** 필수 정보 누락 등으로 인해 신청에 실패(400 에러)했을 때, 서버 응답 본문(`body`)에 고의로 `success: true`가 포함되어 클라이언트가 실패를 성공으로 오인하게 만드는 데이터 불일치 결함이 있습니다.
3. **site075-bug03 (멘토 전용 신청 목록 노출):** 관리자나 멘토만 접근 가능한 전체 신청자 데이터 조회 API에 대해 권한 검증이 누락되어, 일반 사용자가 전체 신청자의 개인정보와 상담 내용을 무단 열람할 수 있는 심각한 보안 취약점이 존재합니다.
