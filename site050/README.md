# LocalTour Guide (site050)

## 개요
- **사이트 이름:** LocalTour Guide
- **포트 번호:** 9379
- **주제:** 지역 관광 코스 추천 서비스

## 주요 기능
- 지역별/테마별 추천 여행 코스 탐색 및 상세 동선 확인
- 경복궁, 남이섬 등 주요 관광지 정보 조회 및 지역 필터링
- 마음에 드는 코스를 나만의 일정으로 저장하고 관리하는 기능
- 방문 순서가 시각화된 타임라인 인터페이스 제공
- 선샤인 옐로우와 오션 블루를 활용한 활기차고 현대적인 여행 플랫폼 UI

## 의도된 오류 (3개)
1. **site050-bug01 (DB 경로 순서 오류):** 코스 상세 조회 시 방문 장소들을 논리적인 방문 순서(`stopOrder`)가 아닌 데이터베이스의 고유 ID 순으로 정렬하여 반환하는 결함이 있어, 동선이 논리적으로 꼬이는 현상이 발생합니다.
2. **site050-bug02 (네트워크 부분 응답):** 관광지 목록 API에서 특정 지역(강원)의 데이터를 요청할 때 상세 설명(`description`) 필드를 누락하여 응답하는 데이터 결함이 있습니다.
3. **site050-bug03 (저장 코스 소유자 검증 누락):** 특정 사용자가 저장한 여행 코스 정보를 조회하는 API에서 소유권 검증 로직이 없어, `savedRouteId` 조작을 통해 타인의 비공개 여행 일정과 방문 계획을 무단 열람할 수 있는 IDOR 보안 취약점이 존재합니다.
