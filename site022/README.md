# Parking Finder (site022)

## 개요
- **사이트 이름:** Parking Finder
- **포트 번호:** 9351
- **주제:** 주차장 검색 및 예약 서비스

## 주요 기능
- 지역명 및 주차장 이름 기반 주차장 검색
- 요금순 정렬 및 잔여 주차 공간 실시간 확인
- 차량 번호 입력을 통한 주차 예약 대행
- 본인의 예약 내역 상세 조회 및 상태 확인
- 타 주차장 서비스 이용 가이드 안내

## 의도된 오류 (3개)
1. **site022-bug01 (DB 위치 필터 오류):** 주차장 검색 시 주소(`address`) 필드를 무시하고 오직 이름(`name`) 필드만 필터링하도록 설계되어, 지역명(예: 강남)으로 검색 시 주소에만 해당 단어가 있는 주차장이 검색되지 않습니다.
2. **site022-bug02 (네트워크 지연 응답):** 주차 예약 신청 API 호출 시 서버에서 의도적으로 약 5초의 지연(`setTimeout`)을 발생시킨 후 응답을 보냅니다.
3. **site022-bug03 (예약 소유자 검증 누락):** 특정 예약 상세 정보를 조회할 때 사용자 권한 검증을 수행하지 않아, `reservationId`만 알면 다른 사람의 차량 번호나 개인 메모 등을 열람할 수 있는 IDOR 취약점이 있습니다.
