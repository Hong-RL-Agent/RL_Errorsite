# ExamTimer (site073)

## 개요
- **사이트 이름:** ExamTimer
- **포트 번호:** 9402
- **주제:** 시험 공부 타이머 및 과목 관리 서비스

## 주요 기능
- 과목별 공부 시간을 실시간으로 측정하고 기록하는 정밀 타이머 기능
- 일일 학습 목표 시간 설정 및 실시간 달성률 시각화 위젯
- 과목 추가, 수정 및 고유 컬러 지정을 통한 체계적인 학습 관리
- 포커스 블루와 라이트 화이트 컬러를 활용한 눈이 편안한 수험생용 UI
- 전체 학습 시간 통계 및 과목별 누적 데이터 분석 대시보드

## 의도된 오류 (3개)
1. **site073-bug01 (DB 공부 시간 계산 오류):** 공부 중 일시정지(Pause)를 했음에도 불구하고, 서버 측 저장 로직 결함으로 인해 일시정지 시간을 차감하지 않고 시작~종료 시간 전체를 공부 시간으로 합산하는 계산 결함이 있습니다.
2. **site073-bug02 (네트워크 중복 저장):** 학습 종료 후 저장 버튼을 연속해서 빠르게 클릭할 경우, 클라이언트와 서버 양측에서 중복 방지 처리가 되어 있지 않아 동일한 학습 기록이 데이터베이스에 여러 번 중복 생성되는 결함이 있습니다.
3. **site073-bug03 (공부 기록 소유자 검증 누락):** 학습 세션 상세 조회 API 요청 시, 요청자가 해당 데이터의 소유자인지 검증하지 않아 `sessionId` 값 조작만으로 다른 사용자의 프라이빗한 공부 기록을 무단 열람할 수 있는 IDOR 보안 취약점이 존재합니다.
