# MiniBank Budget (site061)

## 개요
- **사이트 이름:** MiniBank Budget
- **포트 번호:** 9390
- **주제:** 모의 계좌 예산 관리 서비스

## 주요 기능
- 현재 계좌 잔액 및 계좌번호 확인 등 직관적인 대시보드 제공
- 식비, 급여 등 카테고리별 실시간 거래 내역 필터링 및 상세 내역 모달
- 다른 계좌로의 빠르고 간편한 송금 기능 및 이력 관리
- 로열 블루와 슬레이트 그레이 컬러를 활용한 깔끔하고 세련된 핀테크 UI
- 프리텐다드(Pretendard) 폰트와 볼드한 타이포그래피를 통한 신뢰감 있는 금융 서비스 디자인

## 의도된 오류 (3개)
1. **site061-bug01 (DB 잔액 계산 오류):** 송금 완료 후, 규정에 명시된 이체 수수료(500원)가 계좌 잔액에서 정상적으로 차감되지 않아 실제 자산 현황과 시스템상 잔액이 불일치하는 정산 결함이 있습니다.
2. **site061-bug02 (네트워크 중복 송금):** 송금 버튼을 짧은 시간 내에 여러 번 클릭할 경우, 동일한 송금 요청이 중복으로 처리되어 의도치 않은 이체가 여러 건 발생하는 시스템 결함이 있습니다.
3. **site061-bug03 (계좌 소유자 검증 누락):** 계좌 정보를 조회하는 API에서 소유권 확인 로직이 누락되어, 계좌 ID 조작(`acc-99` 등)을 통해 타인의 자산 현황과 개인 금융 정보를 무단으로 열람할 수 있는 IDOR 보안 취약점이 존재합니다.
