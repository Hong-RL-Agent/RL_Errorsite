# DailyPoll (site044)

## 개요
- **사이트 이름:** DailyPoll
- **포트 번호:** 9373
- **주제:** 설문 및 투표 서비스

## 주요 기능
- IT, 일상, 건강 등 다양한 카테고리의 오늘의 설문 목록 제공
- 클릭 한 번으로 간편하게 참여하는 실시간 투표 프로세스
- 투표 즉시 업데이트되는 통계 그래프 및 결과 보기 기능
- 나의 투표 참여 이력 관리 및 결과 재확인 서비스
- 인디고 블루 톤의 깔끔하고 정돈된 대중 설문 UI

## 의도된 오류 (3개)
1. **site044-bug01 (DB 중복 투표 검증 누락):** 사용자가 특정 설문에 이미 참여했는지 체크하는 로직이 없어, 동일 사용자가 같은 설문에 무제한으로 투표를 반복하여 결과를 왜곡할 수 있는 결함이 있습니다.
2. **site044-bug02 (API 결과 계산 지연):** 특정 설문(예: 점심 메뉴 설문)의 결과를 요청할 때, 서버에서 의도적으로 5초 이상의 응답 지연이 발생하여 로딩이 비정상적으로 길어지는 네트워크 결함이 있습니다.
3. **site044-bug03 (투표 결과 조작 취약점):** 투표 제출 API가 클라이언트에서 전송한 `voteCount` 파라미터를 그대로 현재 득표수에 합산함으로써, 한 번의 요청으로 수천 표를 즉시 조작할 수 있는 보안 취약점이 있습니다.
