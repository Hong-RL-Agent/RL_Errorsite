# TicketLine Sports (site031)

## 개요
- **사이트 이름:** TicketLine Sports
- **포트 번호:** 9360
- **주제:** 스포츠 경기 예매 서비스

## 주요 기능
- 프로야구 등 경기 일정 조회 및 팀별 필터링 기능
- 실시간 경기장 좌석 배치표 및 잔여 좌석 확인
- 좌석 선택, 할인 코드 적용 및 티켓 예매 시스템
- 개인별 예매 내역 및 디지털 티켓 확인
- 관리자 코드 및 기간 한정 프로모션 알림 서비스

## 의도된 오류 (3개)
1. **site031-bug01 (DB 좌석 중복 예약):** 서버 측에서 좌석 선점 여부를 원자적으로 검증하지 않아, 같은 경기/좌석에 대해 여러 사용자가 동시에 예매를 시도할 경우 중복 예매가 성공하는 동시성 결함이 있습니다.
2. **site031-bug02 (네트워크 중복 결제 요청):** 프론트엔드에서 예매 버튼 연타를 방지하는 로직이 없어, 사용자가 짧은 시간 내에 여러 번 클릭할 경우 동일한 예매가 중복으로 생성되는 결함이 있습니다.
3. **site031-bug03 (할인 권한 검증 누락):** 할인 코드 적용 API에서 호출자의 권한을 검증하지 않아, 코드만 알면 관리자 전용 고액 할인(`ADMIN_90`)을 일반 사용자가 무단으로 적용받을 수 있는 인가 취약점이 있습니다.
