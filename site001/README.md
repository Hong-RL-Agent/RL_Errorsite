# Campus Market (site001)

## 개요
- **사이트 이름:** Campus Market
- **사이트 ID:** site001
- **포트 번호:** 9330
- **기술 스택:** Node.js, Express, Vanilla HTML/CSS/JS

## 실행 방법
```bash
cd site001
npm install
npm start
```
서버가 실행되면 브라우저에서 `http://localhost:9330` 으로 접속할 수 있습니다.

## API 엔드포인트 목록
- `GET /api/health` - 서버 상태 확인
- `GET /api/products` - 전체 상품 목록 조회 (필터링 및 정렬 지원)
- `GET /api/products/:id` - 특정 상품 상세 정보 조회
- `POST /api/favorites` - 특정 상품 찜하기(관심 상품 저장)
- `POST /api/login` - 더미 로그인 
- `GET /api/mypage` - 마이페이지 데이터 조회

## 정상 작동 기능 목록 (인터랙션)
1. **상품 목록 조회 및 렌더링** (API 호출)
2. **상품 검색** (제목 기준 클라이언트 필터링)
3. **카테고리 필터링** (API 호출을 통한 필터링)
4. **가격순 정렬** (기본순, 낮은순, 높은순)
5. **상품 상세 모달 오픈** (개별 API 호출)
6. **모달 닫기** (오버레이 및 버튼 클릭)
7. **관심 상품 저장** (API 호출 및 버튼 상태 변경)
8. **내 물건 팔기 폼 제출** (Mock 제출 및 알림)
9. **UI 알림(Notification) 시스템**
10. **거래 후기 탭 전환** (최신순 / 베스트순)
11. **목록 새로고침 버튼** (API 재호출)
12. **화면 네비게이션** (홈 <-> 마이페이지 전환)
