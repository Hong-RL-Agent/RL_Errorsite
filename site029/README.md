# BikeShare Map (site029)

## 개요
- **사이트 이름:** BikeShare Map
- **포트 번호:** 9358
- **주제:** 공유 자전거 대여소 현황 서비스

## 주요 기능
- 지역별 대여소 실시간 현황(대여 가능 자전거 수) 조회 및 검색
- 자전거 수량순 및 기본순(거리순) 정렬 필터
- 개별 대여소 상세 정보(위치, 전체 거치대 수) 확인
- 자전거 대여 신청 및 이용권 기반 대여 로직
- 최근 자전거 이용 내역(대여 장소, 시간, 요금) 조회

## 의도된 오류 (3개)
1. **site029-bug01 (DB 수량 차감 오류):** 사용자가 자전거 대여에 성공했음에도 불구하고, 해당 대여소의 대여 가능 자전거 수(`availableBikes`)가 감소하지 않아 현황판에 잘못된 정보가 표시되는 결함이 있습니다.
2. **site029-bug02 (네트워크 불완전 JSON):** 특정 대여소(st-2)의 상세 정보를 조회할 때, 서버가 구문이 손상된(JSON 닫는 괄호 누락) 응답을 반환하여 클라이언트에서 파싱 에러가 발생하게 만듭니다.
3. **site029-bug03 (인증 없는 대여 요청):** 대여 API 서버 측에서 호출자의 인증 상태를 검증하지 않아, 로그인하지 않은 사용자나 유효하지 않은 토큰을 가진 사용자도 자전거 대여 요청을 성공시킬 수 있는 보안 취약점이 있습니다.
