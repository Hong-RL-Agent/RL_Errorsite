# CraftMarket (site042)

## 개요
- **사이트 이름:** CraftMarket
- **포트 번호:** 9371
- **주제:** 핸드메이드 작품 판매 서비스

## 주요 기능
- 가죽공예, 도자기, 패션 등 다양한 핸드메이드 작품 목록 조회 및 검색
- 카테고리 필터링을 통한 관심 분야 작품 탐색
- 작가에게 직접 요청하는 1:1 맞춤 주문 제작 기능
- 작품 상세 정보 및 작가 프로필 확인
- 내 주문 현황 조회를 통한 제작 진행 상태 추적
- 브라운 & 오트밀 톤의 아날로그 감성 공방 UI

## 의도된 오류 (3개)
1. **site042-bug01 (DB 주문 상태 저장 오류):** 사용자가 주문 제작을 요청하면 상태가 'Requested'로 저장되어야 하나, 내부 로직 오류로 인해 'Draft(임시 저장)' 상태로 저장되어 제작 프로세스가 시작되지 않는 결함이 있습니다.
2. **site042-bug02 (네트워크 실패 상태 코드 오류):** 필수 항목 누락 등으로 주문 제작 요청이 실패했음에도 불구하고, 서버가 에러 코드 대신 `200 OK`를 반환하여 클라이언트의 표준 에러 처리를 방해하는 네트워크 결함이 있습니다.
3. **site042-bug03 (작가 전용 정보 노출):** 작품 상세 조회 API에서 일반 사용자에게 공개되지 않아야 할 작가의 개인 정산 계좌 정보와 내부 관리용 메모가 필터링 없이 노출되는 보안 취약점이 있습니다.
