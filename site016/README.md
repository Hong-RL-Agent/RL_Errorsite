# Budget Pocket (site016)

## 개요
- **사이트 이름:** Budget Pocket
- **포트 번호:** 9345
- **주제:** 개인 가계부 관리 서비스

## 주요 기능
- 지출 내역 기록 및 카테고리 분류
- 월간 지출 요약 및 예산 대비 사용량 시각화
- 소비 분석 리포트 생성
- 지출 내역 필터링 및 정렬 (금액순, 날짜순)
- 사용자 프로필 관리 시뮬레이션

## 의도된 오류 (3개)
1. **site016-bug01 (DB 숫자 타입 처리 오류):** 지출 금액을 숫자형이 아닌 문자열형(`string`)으로 저장하여, 금액순 정렬 시 "10,000" 보다 "2,000"이 더 크게 나오는 등의 사전순 정렬 오류가 발생합니다.
2. **site016-bug02 (API 요청 제한 누락):** 소비 분석 리포트 생성 API에 Rate Limit이 걸려있지 않아, 짧은 시간 동안 무제한으로 API를 호출하여 서버 부하를 유도할 수 있습니다.
3. **site016-bug03 (다른 사용자 데이터 조회):** API 호출 시 `userId` 파라미터만 변경하면 서버 측 검증 없이 다른 사용자의 민감한 지출 내역을 조회할 수 있는 IDOR 취약점이 있습니다.
