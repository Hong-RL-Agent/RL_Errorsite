# PetSitter Match (site034)

## 개요
- **사이트 이름:** PetSitter Match
- **포트 번호:** 9363
- **주제:** 펫시터 예약 서비스

## 주요 기능
- 지역별 전문 펫시터 목록 조회 및 검색
- 펫시터 상세 프로필(평점, 요금, 소개) 확인
- 본인의 반려동물 정보 관리 및 상세 열람
- 날짜 지정을 통한 펫시터 예약 신청 및 내역 확인
- 펫시터 평점순/요금순 정렬 기능

## 의도된 오류 (3개)
1. **site034-bug01 (DB 예약 날짜 검증 누락):** 예약 신청 시 종료일이 시작일보다 빠른(과거인) 경우에도 서버가 이를 차단하지 않고 정상 예약으로 저장하는 데이터 유효성 검증 결함이 있습니다.
2. **site034-bug02 (네트워크 요청 본문 누락):** 예약 시 필수 정보인 반려동물(`petId`)이 선택되지 않은 상태로 요청을 보내도 서버가 오류를 반환하지 않고 성공 응답을 보내는 비즈니스 로직 결함이 있습니다.
3. **site034-bug03 (반려동물 정보 접근 제어 실패):** 특정 반려동물 상세 조회 API에서 작성자 본인 여부를 검증하지 않아, `petId` 조작을 통해 타인의 반려동물 정보와 건강 특이사항 등을 무단으로 열람할 수 있는 IDOR 보안 취약점이 있습니다.
