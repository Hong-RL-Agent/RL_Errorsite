# Museum Pass (site025)

## 개요
- **사이트 이름:** Museum Pass
- **포트 번호:** 9354
- **주제:** 전시회 예매 및 관람 정보 서비스

## 주요 기능
- 현재 전시 및 예정 전시 목록 탐색 및 검색
- 전시회별 작가 정보 및 전시 위치 정보 제공
- 관람 인원 선택 및 티켓 예매 기능
- 개인별 예매 내역(디지털 티켓) 조회 및 관리
- 작가별 상세 소개 및 작품 경향 안내

## 의도된 오류 (3개)
1. **site025-bug01 (DB 티켓 수량 검증 오류):** 티켓 예매 시 서버 측에서 잔여 수량을 검증하지 않아, 실제 남은 티켓보다 더 많은 수량을 예매해도 성공 처리되는 논리적 결함이 있습니다.
2. **site025-bug02 (API 응답 캐시 오류):** 티켓 잔여 수량을 조회하는 API가 실시간 데이터를 반영하지 않고, 항상 고정된 값(50)을 반환하여 사용자가 정확한 잔여석 정보를 파악하지 못하게 합니다.
3. **site025-bug03 (티켓 소유자 검증 누락):** 특정 티켓의 상세 정보를 조회할 때 사용자 권한을 검증하지 않아, `ticketId`만 알면 타인의 예매 정보(이름, 금액, 비밀 메모 등)를 무단으로 열람할 수 있는 IDOR 취약점이 있습니다.
