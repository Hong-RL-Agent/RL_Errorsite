# SmallBiz CRM (site068)

## 개요
- **사이트 이름:** SmallBiz CRM
- **포트 번호:** 9397
- **주제:** 소상공인 고객 관리 서비스

## 주요 기능
- 매출 현황, 오늘의 상담, 총 고객 수 등 비즈니스 핵심 지표 대시보드
- 고객 성함 및 전화번호 기반의 효율적인 고객 검색 및 관리 시스템
- VVIP, 신규, 단골 등 고객 성향에 따른 다중 태그 할당 및 분류
- 프로페셔널 슬레이트와 일렉트릭 블루 컬러를 활용한 현대적인 비즈니스 UI
- 고객 상세 정보 조회 및 상담 이력 기록을 위한 통합 인터페이스

## 의도된 오류 (3개)
1. **site068-bug01 (DB 고객 태그 저장 오류):** 특정 고객의 태그를 수정하여 저장할 때, 서버 측 상태 오염으로 인해 현재 선택한 태그가 아닌 '직전에 수정했던 고객의 태그'가 잘못 저장되는 결함이 있습니다.
2. **site068-bug02 (네트워크 요청 제한 누락):** 서버 리소스를 소모하는 대시보드 통계 API 호출 시 속도 제한(Rate Limit)이 누락되어, 짧은 시간 내에 무제한으로 반복 호출이 가능한 성능 결함이 있습니다.
3. **site068-bug03 (고객 개인정보 노출):** 고객 목록 API 응답 시 화면에 노출되지 않아야 할 고객의 전체 전화번호(`fullPhoneNumber`)와 내부 관리용 비밀 메모(`privateMemo`)가 필터링 없이 그대로 유출되는 보안 취약점이 존재합니다.
