# UsedCar Note (site052)

## 개요
- **사이트 이름:** UsedCar Note
- **포트 번호:** 9381
- **주제:** 중고차 비교 메모 서비스

## 주요 기능
- 그랜저, 제네시스 등 엄선된 중고차 매물 목록 조회 및 검색
- 가격 낮은순/높은순 정렬 및 연식 필터링을 통한 맞춤 차량 탐색
- 마음에 드는 차량을 비교함에 담고 나만의 구매 전략 메모 작성
- 개인별 차량 메모 리스트 관리 및 상세 내용 확인
- 슬레이트 블랙과 실버 컬러를 활용한 고급스러운 프리미엄 자동차 카탈로그 UI

## 의도된 오류 (3개)
1. **site052-bug01 (DB 숫자 정렬 오류):** 차량 가격 정렬 시 숫자가 아닌 문자열 기준으로 비교를 수행하여, `1000`이 `900`보다 먼저 오거나 뒤섞이는 비논리적인 정렬 결함이 있습니다.
2. **site052-bug02 (네트워크 필터 파라미터 무시):** 클라이언트에서 연식 필터(`minYear`)를 설정하여 요청을 보내도 서버 측에서 이를 실제 데이터 필터링 로직에 반영하지 않아 필터링이 무시되는 결함이 있습니다.
3. **site052-bug03 (개인 메모 접근 제어 실패):** 특정 차량 메모를 조회하는 API에서 소유권 검증이 누락되어, `noteId` 조작을 통해 타 사용자가 작성한 비공개 차량 구매 전략이나 협상 메모를 무단으로 열람할 수 있는 IDOR 보안 취약점이 존재합니다.
