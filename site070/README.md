# PuzzleRank (site070)

## 개요
- **사이트 이름:** PuzzleRank
- **포트 번호:** 9339
- **주제:** 퍼즐 풀이 기록 및 랭킹 서비스

## 주요 기능
- 매일 업데이트되는 다양한 난이도의 로직/그리드 퍼즐 풀이 환경 제공
- 실시간 글로벌 랭킹 시스템을 통한 전 세계 플레이어와의 점수 경쟁
- 퍼즐 풀이 성과에 따른 레벨업 및 유니크한 뱃지 수집 컬렉션
- 네온 바이올렛과 사이버 펑크 블루를 활용한 역동적인 게이밍 UI
- 개인 풀이 기록 타임라인 및 상세 퍼즐 성과 분석 기능

## 의도된 오류 (3개)
1. **site070-bug01 (DB 정답 기록 중복):** 동일한 퍼즐에 대해 정답을 여러 번 제출할 경우, 서버 측에서 중복 검증을 수행하지 않아 풀 때마다 점수가 계속 누적되어 랭킹이 비정상적으로 상승하는 결함이 있습니다.
2. **site070-bug02 (네트워크 응답 지연):** 랭킹 시스템에서 'Hard' 난이도 필터를 선택할 경우, 서버 측 로직 결함으로 인해 응답이 8초 이상 지연되어 타임아웃이 발생하거나 사용자 경험을 해치는 성능 결함이 있습니다.
3. **site070-bug03 (점수 조작 취약점):** 정답 제출 시 클라이언트가 계산하여 보낸 `score` 값을 서버가 아무런 검증 없이 그대로 데이터베이스에 반영하여, API 요청 조작만으로 점수를 가공할 수 있는 심각한 보안 취약점이 존재합니다.
