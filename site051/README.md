# QuickSurvey HR (site051)

## 개요
- **사이트 이름:** QuickSurvey HR
- **포트 번호:** 9380
- **주제:** 사내 익명 설문 서비스

## 주요 기능
- 부서별/전사 대상 진행 중인 설문 조사 목록 조회 및 검색
- 주관식 문항에 대한 자유로운 의견 제출 및 익명 보장 인터페이스
- 설문 마감 후 집계된 전체 결과 및 개별 의견 요약 확인
- 나의 설문 참여 현황 관리 및 최근 활동 내역 조회
- 기업형 네이비와 민트 그린 컬러를 활용한 깔끔하고 신뢰감 있는 비즈니스 UI

## 의도된 오류 (3개)
1. **site051-bug01 (DB 익명 처리 실패):** 익명 설문임에도 불구하고 응답 저장 시 사용자의 고유 ID(`userId`)를 데이터베이스에 함께 기록하고, 결과 조회 API 응답 시 개별 답변과 함께 해당 ID를 반환하여 익명성을 파괴하는 결함이 있습니다.
2. **site051-bug02 (네트워크 잘못된 성공 응답):** 이미 응답한 설문에 중복으로 제출을 시도할 경우, 서버는 처리를 거부하면서도 HTTP 상태 코드를 `400`이 아닌 `200 OK`로 반환하여 클라이언트 측 에러 처리를 어렵게 만드는 결함이 있습니다.
3. **site051-bug03 (결과 접근 권한 누락):** 설문이 공식적으로 마감되지 않은 '진행 중(open)' 상태임에도 불구하고, 일반 사용자가 결과 조회 API를 통해 실시간 투표 현황과 타인의 의견을 무단으로 열람할 수 있는 보안 취약점이 있습니다.
