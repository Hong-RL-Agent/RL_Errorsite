# HabitGarden (site077)

## 개요
- **사이트 이름:** HabitGarden
- **포트 번호:** 9406
- **주제:** 습관 기록을 식물 성장으로 표현하는 서비스

## 주요 기능
- 매일의 습관을 완료할 때마다 가상의 식물이 성장하는 시각적 보상 시스템
- 연속 달성일(Streak)에 따른 식물 진화 및 정원 확장 시각화
- 주간/월간 습관 달성률 통계 및 성취 뱃지 수집 시스템
- 보태니컬 그린과 어스 브라운 컬러를 활용한 힐링 테마의 Natural UI
- 습관 체크인, 상세 기록 조회, 뱃지 도감 확인 등 10개 이상의 사용자 인터랙션

## 의도된 오류 (3개)
1. **site077-bug01 (DB 연속 달성 계산 오류):** 습관을 하루 건너뛰더라도 서버 측 스트릭 계산 로직 결함으로 인해 연속 달성 기록이 초기화되지 않고 그대로 유지되거나 증가하는 결함이 있습니다.
2. **site077-bug02 (네트워크 중복 체크인):** 완료 체크 버튼을 빠르게 여러 번 클릭할 경우, 동일한 날짜에 대해 중복 검증을 수행하지 않아 여러 번의 체크인 기록이 중복 저장되는 결함이 있습니다.
3. **site077-bug03 (습관 기록 소유자 검증 누락):** 특정 습관의 상세 정보 및 과거 기록을 조회할 때 요청자가 해당 습관의 소유자인지 확인하는 절차가 누락되어, `habitId` 값 조작만으로 타인의 습관 데이터를 무단 열람할 수 있는 보안 취약점이 존재합니다.
