# BookNest Library (site002)

## 개요
- **사이트 이름:** BookNest Library
- **사이트 ID:** site002
- **포트 번호:** 9331
- **기술 스택:** Node.js, Express, Vanilla HTML/CSS/JS

## 실행 방법
```bash
cd site002
npm install
npm start
```
서버가 실행되면 브라우저에서 `http://localhost:9331` 로 접속할 수 있습니다.

## API 엔드포인트 목록
- `GET /api/health` - 서버 상태 확인
- `GET /api/books` - 도서 목록 조회 (장르 필터링 및 정렬 지원)
- `GET /api/books/:id` - 특정 도서 상세 정보 조회
- `POST /api/rentals` - 특정 도서 대여 요청
- `GET /api/user` - 로그인된 사용자(목업)의 대시보드 정보 조회

## 정상 작동 기능 목록 (인터랙션)
1. **도서 목록 조회 및 렌더링** (API 연동)
2. **도서명/저자 검색** (클라이언트 필터링)
3. **장르 필터링** (API 호출)
4. **목록 정렬 (최신순/평점순)** (API 호출)
5. **도서 상세 모달 오픈** (개별 도서 API 호출)
6. **모달 닫기** (오버레이 및 버튼)
7. **대여 신청 폼 제출** (기간 선택 및 API 호출)
8. **내 서재(대시보드) 화면 전환**
9. **사용자 정보 새로고침** (API 호출)
10. **반납 예정 알림 확인** (토스트 메시지)
11. **추천 도서 새로고침 버튼**
12. **공통 토스트(Toast) 알림 시스템**

## 의도된 오류 (3개)
이 프로젝트는 강화학습 모델(PPO) 훈련을 위해 다음 3개의 오류를 포함하고 있습니다.
1. **site002-bug01 (DB 오류):** DB 정렬 기준 오류 (rating 대신 createdAt으로 정렬)
2. **site002-bug02 (네트워크 오류):** 잘못된 HTTP 상태 코드 (대여 실패 시에도 200 반환)
3. **site002-bug03 (보안 오류):** 민감정보 응답 노출 (passwordHash, internalToken)
