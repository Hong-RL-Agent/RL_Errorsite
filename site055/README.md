# MovieClub Seat (site055)

## 개요
- **사이트 이름:** MovieClub Seat
- **포트 번호:** 9384
- **주제:** 소규모 영화 모임 좌석 예약 서비스

## 주요 기능
- 인터스텔라, 라라랜드 등 상영 영화 목록 조회 및 날짜별 필터링
- 실시간 좌석 배치도를 통한 원하는 좌석 선택 및 예매 기능
- 나의 예매 내역 관리 및 예매 상세 정보 확인
- 미드나잇 블랙과 시네마 레드를 활용한 감각적인 시네마틱 UI
- 다크 모드 기반의 몰입감 넘치는 인터페이스 및 토스트 알림 시스템

## 의도된 오류 (3개)
1. **site055-bug01 (DB 좌석 상태 오류):** 좌석 예약이 성공적으로 완료되어도, 해당 좌석의 상태(`status`)가 `occupied`로 업데이트되지 않아 타 사용자에게 여전히 선택 가능한 상태로 노출되는 정합성 결함이 있습니다.
2. **site055-bug02 (네트워크 중복 요청 처리 실패):** 예약 요청 시 동일 사용자가 동일 좌석을 중복으로 예약하는지 검증하지 않아, 네트워크 지연 등으로 요청이 재전송될 경우 같은 예약 레코드가 여러 개 생성되는 결함이 있습니다.
3. **site055-bug03 (예약 정보 접근 제어 실패):** 특정 예매 내역을 조회하는 API에서 소유권 검증 로직이 누락되어, `reservationId` 조작을 통해 타인의 사적인 영화 예매 정보를 무단으로 열람할 수 있는 IDOR 보안 취약점이 존재합니다.
