# ClinicDesk (site019)

## 개요
- **사이트 이름:** ClinicDesk
- **포트 번호:** 9348
- **주제:** 병원 진료 예약 안내 서비스

## 주요 기능
- 진료과 선택 및 해당 진료과 의사 리스트 조회
- 의사별 상세 정보 확인 및 진료 가능 시간 탐색
- 날짜 및 시간 선택을 통한 진료 예약 신청
- 환자 본인의 예약 내역 실시간 조회
- 예약 단계별 상태 인디케이터 표시

## 의도된 오류 (3개)
1. **site019-bug01 (DB 예약 시간 충돌):** 서버 측에서 동일한 의사 및 동일한 시간대에 대한 중복 예약 검증 로직이 누락되어, 하나의 시간대에 여러 명의 환자가 동시에 예약되는 '더블 부킹'이 허용됩니다.
2. **site019-bug02 (네트워크 실패 재시도 누락):** 의사 목록 조회 API가 간헐적으로 실패(500 에러)할 때, 클라이언트 측에 재시도 로직이 없어 사용자에게 빈 목록이나 오류 메시지만 출력되고 서비스가 중단됩니다.
3. **site019-bug03 (개인정보 과다 노출):** 내 예약 조회 API 응답 데이터에 환자의 민감한 정보인 `patientPrivateCode`(가상 주민번호)가 포함되어 외부로 노출되는 보안 취약점이 있습니다.
