# SalonBook (site039)

## 개요
- **사이트 이름:** SalonBook
- **포트 번호:** 9368
- **주제:** 미용실 예약 서비스

## 주요 기능
- 스타일리스트별(실장, 부원장 등) 프로필 조회 및 시술 전문 분야 확인
- 컷, 펌, 염색 등 살롱 시술 메뉴 선택 및 가격 정보 제공
- 날짜 및 시간 선택을 통한 실시간 예약 신청 프로세스
- 내 예약 현황 조회를 통한 예약 내역 관리 및 상세 정보 확인
- 프리미엄 살롱 브랜드 이미지에 맞춘 고급스러운 골드 & 화이트 UI

## 의도된 오류 (3개)
1. **site039-bug01 (DB 예약 가능 시간 오류):** 스타일리스트의 예약 가능 시간을 조회할 때, 이미 다른 사용자가 점유한 시간대를 필터링하지 않고 전체 목록을 반환하여 중복 예약이 가능한 데이터 로직 결함이 있습니다.
2. **site039-bug02 (API 요청 중복 처리 실패):** 예약 신청 버튼 클릭 시 서버 측에서 멱등성 검증을 하지 않아, 네트워크 지연 등으로 인한 버튼 연타 시 동일한 예약이 여러 개 생성되는 네트워크 결함이 있습니다.
3. **site039-bug03 (예약 상세 권한 검증 누락):** 예약 상세 정보 조회 API에서 요청자가 실제 소유자인지 확인하지 않아, `bookingId` 조작을 통해 타인의 시술 내역, 시간, 관리자 메모 등을 무단으로 열람할 수 있는 IDOR 보안 취약점이 있습니다.
