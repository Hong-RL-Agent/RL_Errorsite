# MealPrep Calendar (site067)

## 개요
- **사이트 이름:** MealPrep Calendar
- **포트 번호:** 9396
- **주제:** 식단 준비 캘린더 서비스

## 주요 기능
- 아침, 점심, 저녁 등 일일 식단 기록 및 주간 캘린더 기반 관리 기능
- 섭취한 음식의 칼로리를 자동 합산하여 일일 목표 대비 현황 제공
- 레시피 탐색 및 식단 준비를 위한 장보기 체크리스트 관리
- 프레시 민트와 라이트 우드 컬러를 활용한 건강하고 정갈한 헬스케어 UI
- 날짜별 빠른 이동 및 직관적인 칼로리 요약 대시보드

## 의도된 오류 (3개)
1. **site067-bug01 (DB 칼로리 합계 오류):** 대시보드에서 일일 총 칼로리를 계산할 때, 사용자가 삭제 처리한 식단 항목(`isDeleted: true`)까지 모두 합계에 포함하여 실제보다 높게 표시되는 계산 결함이 있습니다.
2. **site067-bug02 (네트워크 응답 순서 문제):** 사용자가 날짜를 빠르게 변경할 경우, 이전 요청에 대한 서버 응답이 뒤늦게 도착하여 최신 날짜의 화면을 이전 날짜의 데이터로 덮어씌우는 네트워크 레이스 컨디션 결함이 있습니다.
3. **site067-bug03 (사용자 식단 접근 제어 실패):** 식단 목록 조회 시 현재 로그인한 사용자의 데이터만 필터링하지 않고, 데이터베이스 내 모든 사용자의 식단 기록을 함께 반환하는 심각한 보안 데이터 유출 취약점이 존재합니다.
