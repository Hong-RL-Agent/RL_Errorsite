# NewsBrief (site020)

## 개요
- **사이트 이름:** NewsBrief
- **포트 번호:** 9349
- **주제:** 개인화 뉴스 큐레이션 서비스

## 주요 기능
- 최신 뉴스 목록 조회 및 카테고리별 필터링
- 기사 검색 및 상세 내용 보기 (모달)
- 관심 기사 북마크(저장) 기능
- 실시간 트렌드 키워드 확인 및 갱신
- 사용자 구독 설정 및 관심 카테고리 관리
- 기사 읽음 처리(Read/Unread) 상태 관리

## 의도된 오류 (3개)
1. **site020-bug01 (DB 읽음 상태 오류):** 기사를 클릭하여 읽음 처리할 때, 서버가 전달받은 기사 ID가 아닌 무조건 목록의 마지막 기사의 상태를 '읽음'으로 변경하는 논리적 오류가 있습니다.
2. **site020-bug02 (오래된 API 응답):** 트렌드 키워드 조회 API가 최신 실시간 데이터를 계산하지 않고, 항상 2020년도의 고정된 낡은 mock 데이터를 반환합니다.
3. **site020-bug03 (구독 설정 권한 검증 누락):** 구독 설정 조회 API에서 사용자의 권한을 검증하지 않아, `subscriptionId`만 알면 타인의 민감한 개인 설정 정보를 무단으로 조회할 수 있는 IDOR 취약점이 있습니다.
