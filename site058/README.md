# CafeStudy Pass (site058)

## 개요
- **사이트 이름:** CafeStudy Pass
- **포트 번호:** 9387
- **주제:** 스터디카페 좌석 이용권 관리 서비스

## 주요 기능
- 실시간 좌석 현황 확인 및 원하는 좌석(Focus, Cafe, Private Zone) 예약 기능
- 시간권 및 기간권 등 다양한 스터디카페 이용권 탐색 및 구매
- 나의 이용권 목록 조회 및 잔여 시간, 만료일 상세 정보 확인
- 코지 우드와 딥 포레스트 그린 컬러를 활용한 아늑하고 몰입감 있는 스터디카페 UI
- 다크 세리프 타이포그래피와 나무 질감의 레이아웃을 통한 전문적인 관리 서비스 디자인

## 의도된 오류 (3개)
1. **site058-bug01 (DB 이용권 만료 검증 오류):** 좌석 예약 시 선택한 이용권의 유효기간이 이미 지났음에도 불구하고, 서버에서 이를 정상적인 이용권으로 판단하여 예약을 승인하는 검증 결함이 있습니다.
2. **site058-bug02 (네트워크 응답 상태 불일치):** 이용권 구매 과정에서 결제 실패(예: 잔액 부족)가 발생하더라도, 서버가 HTTP 상태 코드를 `201 Created`로 반환하여 클라이언트가 구매에 성공한 것으로 오해하게 만드는 결함이 있습니다.
3. **site058-bug03 (이용권 소유자 검증 누락):** 특정 이용권의 상세 정보를 조회하는 API에서 소유권 확인 로직이 누락되어, `passId` 조작을 통해 타인의 결제 내역이나 잔여 시간 정보를 무단으로 열람할 수 있는 IDOR 보안 취약점이 존재합니다.
