# LocalFarm Direct (site014)

## 개요
- **사이트 이름:** LocalFarm Direct
- **포트 번호:** 9343
- **주제:** 로컬 농산물 직거래 서비스

## 주요 기능
- 로컬 농산물 목록 조회 및 지역별 필터링
- 농산물 장바구니 담기 및 주문 결제
- 공동구매 참여 및 실시간 달성률 확인
- 농가 정보 및 상세 페이지 뷰
- 신선도 리뷰 확인

## 의도된 오류 (3개)
1. **site014-bug01 (DB 재고 검증 누락):** 공동구매 참여 시 요청한 수량이 현재 남은 재고보다 많아도 검증 없이 승인되어 재고가 음수(-)가 되는 논리적 결함이 있습니다.
2. **site014-bug02 (서버 응답 포맷 오류):** 주문 성공 시 클라이언트가 기대하는 JSON 형식이 아닌 HTML 문자열을 반환하여 프론트엔드 파싱 에러를 유발합니다.
3. **site014-bug03 (민감한 내부 경로 노출):** 서버 측 에러 발생 시 응답 객체에 서버의 절대 경로와 내부 변수명이 포함되어 민감한 정보가 외부로 유출됩니다.
