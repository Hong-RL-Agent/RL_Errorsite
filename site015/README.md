# EventBridge (site015)

## 개요
- **사이트 이름:** EventBridge
- **포트 번호:** 9344
- **주제:** 지역 행사 탐색 및 참가 신청 서비스

## 주요 기능
- 지역 행사(문화, 비즈니스, 교육 등) 탐색 및 카테고리 필터링
- 행사 상세 정보 조회 및 참가 신청 시스템
- 행사 공지사항 게시판 확인
- 본인의 참가 신청 내역 조회
- 관리자 권한을 통한 전체 신청자 명단 조회 (테스트용)

## 의도된 오류 (3개)
1. **site015-bug01 (DB 카운트 불일치):** 참가 신청이 완료되어도 해당 행사의 잔여석(`remainingSeats`) 숫자가 줄어들지 않아, 무한정 신청이 가능해지는 논리적 오류가 있습니다.
2. **site015-bug02 (캐시된 오래된 응답):** 행사 목록 API에 1시간의 강력한 캐시(`Cache-Control`)가 설정되어 있어, 신청 후에도 목록으로 돌아오면 정보가 즉시 갱신되지 않는 현상이 발생합니다.
3. **site015-bug03 (관리자 기능 인증 우회):** URL에 `?admin=true` 파라미터만 붙이면 서버가 인증 없이 모든 사용자의 신청 정보를 반환하는 보안 취약점이 있습니다.
