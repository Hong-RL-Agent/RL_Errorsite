# Smart Pantry (site030)

## 개요
- **사이트 이름:** Smart Pantry
- **포트 번호:** 9359
- **주제:** 냉장고 재료 및 유통기한 관리 서비스

## 주요 기능
- 냉장고 내 식재료 목록 관리 및 카테고리별 분류
- 유통기한 임박 재료 자동 감지 및 알림 기능
- 보유 재료 기반 추천 레시피 탐색
- 부족한 재료 장보기 목록 추가 및 관리
- 재료별 수량 파악 및 유통기한순 정렬 기능

## 의도된 오류 (3개)
1. **site030-bug01 (DB 유통기한 비교 오류):** 임박 재료 조회 API에서 유통기한이 오늘 이전인(이미 만료된) 재료를 걸러내지 못하고 임박 목록에 함께 표시하는 데이터 쿼리 로직 결함이 있습니다.
2. **site030-bug02 (요청 본문 타입 불일치):** 새로운 재료를 추가할 때 수량(`quantity`) 필드를 숫자형이 아닌 문자열로 전송해도 서버가 이를 검증 없이 그대로 저장하여 데이터 타입 불일치를 야기하는 결함이 있습니다.
3. **site030-bug03 (사용자별 데이터 분리 실패):** 냉장고 목록 조회 시 사용자별 권한 검증과 필터링이 누락되어, 시스템 내 모든 사용자의 식재료 데이터를 한꺼번에 반환하는 심각한 멀티테넌시 보안 취약점이 있습니다.
