# Cafe Queue (site012)

## 개요
- **사이트 이름:** Cafe Queue
- **포트 번호:** 9341
- **주제:** 카페 모바일 주문 및 대기열 서비스

## 주요 기능
- 카페 메뉴 조회 및 카테고리 필터링
- 메뉴 옵션(온도) 선택 및 장바구니 담기
- 실시간 주문 대기열 조회
- 주문 상태 확인 (내 주문 번호 및 진행 상황)
- 멤버십 포인트 조회

## 의도된 오류 (3개)
1. **site012-bug01 (DB 자동 증가 키 충돌):** 주문 생성 시 ID 생성 로직의 결함으로 인해 기존 주문과 동일한 ID가 부여되어 데이터 충돌이 발생할 수 있습니다.
2. **site012-bug02 (네트워크 중복 제출):** 주문 버튼 클릭 시 서버에서 중복 요청을 처리하지 않아 광클 시 동일 주문이 여러 번 생성됩니다.
3. **site012-bug03 (포인트 권한 검증 누락):** 타인의 `memberId`를 알면 해당 사용자의 멤버십 포인트 정보를 조회할 수 있는 IDOR 취약점이 있습니다.
