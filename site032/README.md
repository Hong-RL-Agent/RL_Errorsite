# TutorMatch (site032)

## 개요
- **사이트 이름:** TutorMatch
- **포트 번호:** 9361
- **주제:** 과외 선생님 매칭 서비스

## 주요 기능
- 과목별(수학, 영어, 과학, 코딩 등) 전문 선생님 탐색 및 검색
- 선생님별 상세 프로필(경력, 자기소개, 평점, 수업료) 확인
- 가격순/기본순 정렬 필터링을 통한 맞춤형 검색
- 비대면 상담 신청 기능 및 신청 현황 확인
- 실시간 추천 알고리즘 기반의 오늘의 추천 선생님 섹션

## 의도된 오류 (3개)
1. **site032-bug01 (DB 가격 정렬 오류):** 선생님 목록을 '가격 낮은순'으로 정렬할 때, 수업료(`hourlyRate`)를 숫자가 아닌 문자열 기준으로 정렬하여 상식적이지 않은 순서로 목록이 표시되는 결함이 있습니다.
2. **site032-bug02 (API 필드명 불일치):** 추천 선생님 조회 API에서 클라이언트가 기대하는 `tutorName` 필드 대신 `name` 필드로 데이터를 반환하여, 화면에 추천 선생님의 이름이 "알 수 없음"으로 표시되는 스키마 불일치 오류가 있습니다.
3. **site032-bug03 (상담 신청 인증 누락):** 상담 신청 API에서 호출자의 인증 상태를 검증하지 않아, 로그인하지 않은 익명 사용자나 유효하지 않은 토큰을 가진 사용자도 무단으로 상담 신청을 생성할 수 있는 보안 취약점이 있습니다.
