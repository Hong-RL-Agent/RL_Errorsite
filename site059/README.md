# IndieGame Shelf (site059)

## 개요
- **사이트 이름:** IndieGame Shelf
- **포트 번호:** 9388
- **주제:** 인디게임 탐색 및 찜 서비스

## 주요 기능
- 네온 러너, 픽셀 퀘스트 등 개성 넘치는 인디게임 목록 탐색 및 상세 정보 확인
- 장르별 필터링(액션, 어드벤처, 전략 등) 및 평점 기반 게임 큐레이션
- 출시 예정인 기대작들의 라인업 확인 및 출시 일정 트래킹
- 마음에 드는 게임을 찜 목록에 추가하고 나만의 위시리스트 관리
- 픽셀 퍼플과 사이버 옐로우 컬러를 활용한 감각적이고 힙한 레트로 게이밍 UI

## 의도된 오류 (3개)
1. **site059-bug01 (DB 평점 평균 오류):** 게임의 평균 평점을 계산할 때, 시스템상에서 삭제된 리뷰(`isDeleted: true`)까지 모두 합산에 포함하여 계산하는 집계 결함이 있습니다.
2. **site059-bug02 (네트워크 스키마 누락):** 출시 예정 게임 API에서 특정 항목의 출시일(`releaseDate`) 필드가 누락되어 전송됨으로써, 클라이언트에서 날짜 정보가 정상적으로 표시되지 않는 데이터 결함이 있습니다.
3. **site059-bug03 (찜 목록 사용자 검증 누락):** 찜하기 요청 시 서버가 요청 본문의 `userId`를 현재 사용자 세션과 대조하지 않아, 타인의 ID로 찜 목록에 게임을 강제로 추가할 수 있는 보안 취약점이 존재합니다.
